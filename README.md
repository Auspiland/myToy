## geo_map
- **설명**: 사각형 영역 또는 좌표를 대한민국(남한/북한) 행정구역(시/도/시군구 등)으로 변환하고 GeoJSON으로 출력하는 지도 변환 도구입니다.
- **기술**: Python, GeoJSON, Shapefile, STRtree 기반 공간 인덱스, GitHub 연동.
- **특징**: 사각형→행정구역 변환 및 다수 배치 처리, 섬 자동 감지(섬 특화 표기), 행정구역 경계 조회 및 GeoJSON 생성/시각화.

## Mini-Pipeline
- **설명**: YouTube 스크립트 추출부터 문장 전처리, Kafka/Opensearch 업로드까지의 Docker 기반 스트리밍 E2E 파이프라인입니다.
- **기술**: Docker/Compose, Apache Kafka, Apache Spark (Structured Streaming), FastAPI, Nginx, Opensearch, Kiwipiepy, Selenium(수집 보조).
- **특징**: 컨테이너화된 스트리밍 파이프라인(Kafka↔Spark↔Opensearch), Nginx/FASTAPI 기반 웹 라우팅 및 검색 UI, Kiwipiepy 기반 한국어 전처리(udf) 및 룰베이스 보정.

## B_project
- **설명**: 알고리즘 문제 풀이 및 코딩 테스트 관련 유틸·연습 리포지토리입니다.
- **기술**: Python 중심(자료구조·그래프·트리·문자열·고급 알고리즘), Jupyter 노트북, FastAPI/Async 관련 가이드 문서 등.
- **특징**: boj_bible 알고리즘 라이브러리(기본/그래프/트리/문자열/고급), CT 자동화·기록 도구 및 LLM 호출 유틸(기본 모델이 gpt-5-nano로 변경), kakao_history 노트북에 포함된 실전 풀이 유틸 함수들.
