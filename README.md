## geo_map
- **설명**: 좌표 또는 사각형 영역을 대한민국/북한 행정구역(시·도 등)으로 변환 및 시각화하는 도구입니다.
- **기술**: Python(Shapely/GeoPandas 기반), STRtree 공간색인, GeoJSON 입출력.
- **특징**: 사각형·포인트→행정구역 변환(대표문구/상세), 섬 자동 감지 및 경계 조회, GeoJSON 생성·일괄 변환 지원.

## Mini-Pipeline
- **설명**: YouTube 스크립트 추출부터 문장 전처리·Kafka·OpenSearch 업로드까지의 E2E 스트리밍 파이프라인입니다.
- **기술**: Docker Compose, Apache Spark Structured Streaming, Kafka(kRaft), OpenSearch, FastAPI/Nginx, Kiwipiepy(한국어 전처리).
- **특징**: 마운트 폴더→Spark readStream으로 파일 감지·처리, Kafka 토픽 기반 파이프라인, OpenSearch 인덱싱 및 웹 검색 연동.

## B_project
- **설명**: 알고리즘 문제 풀이·코딩 테스트 지원용 코드 모음 및 도구 모음입니다.
- **기술**: Python 기반 알고리즘 라이브러리(모듈화된 자료구조/그래프/트리 등), Jupyter 노트북, FastAPI 예제 문서.
- **특징**: boj_bible(자료구조·알고리즘 모듈) 제공, CT 자동화·LLM 유틸(기본 모델 변경 등) 및 문제 풀이 예제 노트북 포함.
