## geo_map
- **설명**: 좌표나 사각형 영역을 대한민국(남/북) 행정구역으로 변환하고 시각화하는 도구  
- **기술**: Python, Shapely/GeoPandas(또는 STRtree 기반 공간 인덱스), GeoJSON 출력  
- **특징**: 사각형/좌표→행정구역 변환 및 대표 문구 생성, 섬 자동 감지(방향/면적 규칙), 경계 조회·GeoJSON 생성 및 일괄 변환

## Mini-Pipeline
- **설명**: YouTube 스크립트 추출부터 전처리·Kafka·Opensearch 업로드까지의 E2E 스트리밍 파이프라인  
- **기술**: Docker(다중 docker-compose), Spark Structured Streaming, Kafka(kRaft), OpenSearch, FastAPI/Nginx, Kiwipiepy  
- **특징**: 스트리밍 기반 파일 감시→Kafka 토픽 연계 처리, 문장 전처리(Kiwi + 룰베이스) 및 Opensearch 인덱싱, 컨테이너별 네트워크 분리·운영

## B_project
- **설명**: 알고리즘 문제 풀이·코딩 테스트 지원 툴과 관련 실습 저장소  
- **기술**: Python 중심(라이브러리 모듈 + Jupyter 노트북), 일부 Async/FastAPI 구성 문서 포함  
- **특징**: boj_bible 알고리즘 모듈 모음(자료구조·그래프·트리·문자열 등), CT 자동화 도구(LLM 연동 유틸 포함) 및 문제풀이 노트북·실전 예제
