## geo_map
- **설명**: 사각형 영역이나 좌표를 대한민국(남한/북한) 행정구역 표기로 변환하고 GeoJSON으로 시각화하는 도구입니다.  
- **기술**: Python, Shapely/STRtree 기반 공간 인덱싱, GeoJSON, GitHub 연동 스크립트.  
- **특징**: Rect2Region을 통한 영역→행정구역 변환(대표 표현 포함), 섬 감지 및 경계 조회(get_region_boundary), GeoJSON 일괄 생성·배치 처리(convert_many/convert_json_file).

## Mini-Pipeline
- **설명**: YouTube 영상 스크립트 추출 → 문장 전처리 → Kafka/Opensearch에 업로드하는 E2E 스트리밍 파이프라인입니다.  
- **기술**: Docker(다중 compose), Spark Structured Streaming, Kafka(kRaft), OpenSearch, FastAPI/Nginx, Kiwipiepy(한국어 전처리).  
- **특징**: 컨테이너화된 스트리밍 아키텍처(Producer/Consumer/Spark), Kiwipiepy 기반 오탈자·문장분리 파이프라인(pandas_udf 사용), Opensearch 연동 및 웹 검색 UI 연동.

## B_project
- **설명**: 알고리즘 학습·코딩테스트 풀이용 라이브러리 및 자동화/기록 도구 모음입니다.  
- **기술**: Python(알고리즘 모듈, Jupyter 노트북), FastAPI/Async DB(문서 내 가이드), LLM 유틸(CT 도구 내).  
- **특징**: boj_bible(자료구조·그래프·트리·문자열·고급 알고리즘) 모듈, CT 자동화(전처리·LLM 헬퍼) 및 노트북 예제, 운영·개발 가이드(special_prompt.md)와 모델 기본값 변경 안내.
