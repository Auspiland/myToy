## geo_map
- **설명**: 사각형 영역 또는 좌표를 한국 행정구역(시/도/시군구 수준)으로 변환하고 GeoJSON으로 시각화하는 도구  
- **기술**: Python, geopandas/shapely, STRtree 공간 인덱스, GeoJSON 출력  
- **특징**: 사각형→행정구역 변환(대량 처리 지원), 좌표→행정구역 변환 및 경계 조회, 섬 자동 감지·GeoJSON 생성 및 시각화

## Mini-Pipeline
- **설명**: YouTube 스크립트 추출부터 문장 전처리 및 Kafka/Opensearch 업로드까지의 E2E 스트리밍 파이프라인  
- **기술**: Docker (compose), Spark Structured Streaming, Kafka(kRaft), Opensearch, FastAPI, Nginx, Selenium, Kiwipiepy  
- **특징**: 컨테이너화된 스트리밍 파이프라인(입력 → Spark → Kafka → Opensearch), Kiwi 기반 텍스트 전처리(pandas_udf 병렬 처리), 웹 검색/조회용 FastAPI + Nginx 연동

## B_project
- **설명**: 알고리즘 문제 풀이용 라이브러리 및 코딩 테스트 자동화 도구 모음  
- **기술**: Python (알고리즘 모듈/노트북), FastAPI/Async 스택 설명 문서(운영 가이드)  
- **특징**: boj_bible(자료구조·알고리즘 모듈)과 CT 자동화 도구, LLM 유틸(기본 모델 gpt-5 → gpt-5-nano 변경 안내) 및 문제 풀이 예제 노트북 첨부
