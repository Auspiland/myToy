## geo_map
- **설명**: 사각형 영역 또는 좌표를 대한민국(남한/북한) 행정구역으로 변환하고 경계/GeoJSON을 생성·시각화하는 도구
- **기술**: Python, Shapely/STRtree 기반 공간 인덱스, GeoJSON, GeoPandas(데이터 처리), LLM(지명 번역)
- **특징**: 사각형→행정구역 변환(고성능 v2), 좌표→행정구역 매핑 및 행정구역 경계 조회, 섬 자동 감지·GeoJSON 생성 및 일괄 변환

## Mini-Pipeline
- **설명**: YouTube URL로부터 스크립트를 추출해 문장 전처리 후 Kafka·Opensearch에 업로드하는 E2E 스트리밍 파이프라인
- **기술**: Docker(다중 컨테이너, docker-compose), Spark Structured Streaming, Kafka(kRaft), Opensearch, FastAPI, Nginx, Kiwipiepy
- **특징**: 컨테이너 기반 네트워크 아키텍처(웹→Nginx→FastAPI), Spark로 스트리밍 처리·Kafka 토픽 연동·Opensearch 저장, Kiwipiepy+pandas_udf를 이용한 한국어 전처리 파이프라인

## B_project
- **설명**: 알고리즘 문제 풀이 및 코딩 테스트 자동화·기록용 모음 프로젝트
- **기술**: Python 기반 알고리즘 모듈 및 스크립트, Jupyter 노트북, FastAPI/Async 스택 문서(운영 가이드)
- **특징**: boj_bible(자료구조·그래프·트리·문자열·고급 알고리즘) 라이브러리, CT 자동화 도구 및 LLM 헬퍼 함수(기본 모델 변경 안내 포함), kakao_history.ipynb 등 실전 문제 풀이 자원
