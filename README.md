## geo_map
- **설명**: 좌표 또는 사각형 영역을 대한민국 행정구역(시/도/시군구 등)으로 매핑하고 GeoJSON 생성·시각화하는 도구입니다.  
- **기술**: Python, Shapely/STRtree, GeoJSON/GeoPandas 기반 공간 처리.  
- **특징**: 사각형→행정구역 대량 변환(대표 텍스트 옵션 포함), 좌표→행정구역 변환 및 섬 자동 감지, 경계 조회·GeoJSON 생성 및 시각화 기능.

## Mini-Pipeline
- **설명**: YouTube 스크립트 추출부터 문장 전처리·Kafka 전송·OpenSearch 인덱싱까지 수행하는 컨테이너화된 스트리밍 E2E 파이프라인입니다.  
- **기술**: Docker Compose, Spark Structured Streaming, Kafka(kRaft), OpenSearch, FastAPI/Nginx, Kiwipiepy(한국어 전처리).  
- **특징**: 파일 감시→Kafka 토픽 흐름 기반 스트리밍 처리, Kiwipiepy+pandas_udf를 이용한 한국어 전처리 및 룰베이스 정제, 웹 UI/검색 연동(Nginx+FastAPI).

## B_project
- **설명**: 알고리즘 문제 풀이 및 코딩 테스트 학습·자동화용 라이브러리와 예제 모음입니다.  
- **기술**: Python 기반 알고리즘 모듈(그래프·트리·문자열·네트워크 플로우 등), Jupyter 노트북, 관련 개발 문서.  
- **특징**: boj_bible(자료구조·알고리즘 구현 모듈), CT(코딩 테스트) 자동화·LLM 헬퍼 유틸리티(기본 모델 변경 알림 포함), 실습용 노트북·운영·개발 가이드 문서.
