## geo_map
- **설명**: 좌표 또는 사각형 영역을 한국(남·북) 행정구역 단위로 변환하고 GeoJSON 생성 및 시각화하는 도구입니다.
- **기술**: Python, Shapely/STRtree 공간 인덱스, GeoPandas/GeoJSON, 공간 데이터(GeoJSON) 처리 유틸.
- **특징**: Rect2Region 기반 사각형→행정구역 변환(대표 텍스트/상세), 대량 변환 성능 최적화(convert_many), 경계 조회·섬 자동 감지 및 GeoJSON 출력.

## Mini-Pipeline
- **설명**: YouTube 스크립트 추출부터 전처리・Kafka・Spark・Opensearch 업로드까지 컨테이너화된 스트리밍 E2E 파이프라인입니다.
- **기술**: Docker Compose, Apache Spark(Structured Streaming), Kafka(kRaft), Opensearch, FastAPI, Nginx, Selenium, Kiwipiepy.
- **특징**: 컨테이너 기반 스트리밍 처리(파일→Kafka→Spark→Opensearch), 한국어 전처리(Kiwipiepy + 룰베이스)와 Timeout/Trigger 기반 스트리밍 제어, 웹 검색 인터페이스 연동.

## B_project
- **설명**: 알고리즘 문제 풀이(코딩 테스트)용 라이브러리 및 도구 모음입니다.
- **기술**: Python, Jupyter Notebook, 알고리즘/자료구조 구현(그래프·트리·문자열 등), LLM 연동 유틸.
- **특징**: boj_bible(자료구조·알고리즘 모듈) 제공, CT 자동화·예제 노트북(kakao_history.ipynb) 포함, LLM 관련 기본 모델 파라미터 변경 및 운영 가이드(special_prompt.md).
