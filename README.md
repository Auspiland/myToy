## geo_map
- **설명**: 사각형 영역이나 좌표를 대한민국(남한/북한) 행정구역(시/도/구/리 등)으로 변환하고 GeoJSON으로 시각화하는 도구입니다.  
- **기술**: Python, Shapely/GeoPandas 기반 공간 처리, STRtree 공간 인덱스, GeoJSON.  
- **특징**: 사각형→행정구역 변환(대량 처리 최적화), 섬 자동 감지 및 방향 기반 대표 표현, 경계 조회·GeoJSON 생성·일괄 JSON 변환.

## Mini-Pipeline
- **설명**: YouTube 스크립트 추출 → 전처리 → Kafka/Spark로 스트리밍 파이프라인을 거쳐 Opensearch에 업로드하고 웹에서 검색·조회하는 E2E 컨테이너화 파이프라인입니다.  
- **기술**: Docker/Docker Compose, Apache Spark(Structured Streaming), Kafka(kRaft), Opensearch, FastAPI, Nginx, Selenium, Kiwipiepy(형태소), Python.  
- **특징**: 스트리밍 기반 자동화 파이프라인, Spark와 Kafka 연동으로 대용량 처리·파이프라인 모니터링, Opensearch 연동 검색/웹 노출.

## B_project
- **설명**: 알고리즘 문제 풀이용 라이브러리(boj_bible)와 코딩 테스트 자동화·기록 도구 및 참고 노트북을 포함한 리포지토리입니다.  
- **기술**: Python(알고리즘 구현, Jupyter Notebook), CT 파이프라인 관련 문서에는 FastAPI/Async, PostgreSQL(Async), Redis, Docker/CI 가이드 포함.  
- **특징**: 자료구조·그래프·트리·문자열·고급 알고리즘 모듈화(basic/graph/tree/string/advanced), CT 자동화 도구·예제 노트북 제공, LLM 관련 함수 기본 모델이 gpt-5 → gpt-5-nano로 변경된 공지 포함.
