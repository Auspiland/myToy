## geo_map
- **설명**: 사각형 영역 또는 좌표를 대한민국(남한/북한) 행정구역으로 매핑하고 GeoJSON으로 출력/시각화하는 도구입니다.
- **기술**: Python, Shapely/STRtree 기반 공간 색인, GeoJSON, GitHub 자동 푸시 스크립트
- **특징**: 사각형→행정구역 변환(대량 처리·성능 최적화), 좌표→지역 변환 및 경계 조회, 섬 자동 감지·GeoJSON 생성 지원

## Mini-Pipeline
- **설명**: YouTube 스크립트 추출 → 문장 전처리 → Kafka/Spark 스트리밍을 거쳐 Opensearch에 적재하는 E2E 파이프라인입니다.
- **기술**: Docker(Compose), Spark Structured Streaming, Kafka(kRaft), Opensearch, FastAPI/Nginx, Kiwipiepy, Selenium(수집 보조)
- **특징**: 컨테이너화된 멀티 서비스 아키텍처, Streaming 기반 데이터 흐름(Kafka↔Spark↔Opensearch), 한국어 전처리(키위·룰베이스)와 검색 UI 연동

## B_project
- **설명**: 알고리즘 문제 풀이 라이브러리 및 코딩 테스트 자동화·기록 도구 모음입니다.
- **기술**: Python, Jupyter Notebook, FastAPI(문서/운영 가이드 참조), LLM 호출 유틸리티
- **특징**: BOJ용 알고리즘 모듈(자료구조·그래프·트리·문자열 등), CT 자동화 도구(LLM 기본 모델 변경 알림 포함), 문제 풀이 예제 노트북 제공
