## geo_map
- **설명**: 사각형 영역 또는 좌표를 대한민국(남한/북한) 행정구역으로 매핑하고 GeoJSON으로 출력·시각화하는 공간 변환 도구입니다.  
- **기술**: Python, Shapely/STRtree 기반 공간 인덱스, GeoJSON 처리.  
- **특징**: 사각형→행정구역(대량 처리 최적화), 좌표→행정구역 변환 및 섬 자동 감지, 경계 조회·GeoJSON 생성/시각화.

## Mini-Pipeline
- **설명**: YouTube 스크립트 추출부터 문장 전처리, Kafka/Opensearch 업로드까지의 컨테이너화된 스트리밍 E2E 파이프라인입니다.  
- **기술**: Docker(Compose), Spark Structured Streaming, Kafka(kRaft), Opensearch, FastAPI/Nginx, Kiwipiepy, Selenium(수집 보조).  
- **특징**: Spark+Kafka 기반 실시간 파이프라인, Kiwipiepy·룰 기반 텍스트 전처리 및 오타·문장 분리, Docker 멀티컨테이너 아키텍처로 배포·연동 구성.

## B_project
- **설명**: 알고리즘 문제 풀이(백준)와 코딩 테스트 자동화·기록을 위한 도구 모음 프로젝트입니다.  
- **기술**: Python 기반 알고리즘 라이브러리, Jupyter 노트북, LLM 연동 유틸(FastAPI 연계 가능).  
- **특징**: boj_bible(자료구조·그래프·트리·문자열·고급 알고리즘) 모듈, CT 자동화·LLM 유틸(기본 모델 파라미터 변경 안내 포함), 예제 노트북(kakao_history.ipynb) 제공.
