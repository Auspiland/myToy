## geo_map
- **설명**: 좌표나 사각형 영역을 한국 행정구역(시/도/군/구 등)으로 자동 변환하고 GeoJSON으로 출력하는 도구입니다.  
- **기술**: Python, Shapely/STRtree 기반 공간 인덱싱, GeoJSON, 행정경계 데이터(GeoJSON)  
- **특징**: 사각형→행정구역 변환(대표 표현 포함), 좌표→행정구역 매핑 및 경계 조회, 섬 자동 감지·GeoJSON 생성 및 일괄 변환

## Mini-Pipeline
- **설명**: YouTube 스크립트 추출 후 문장 전처리→Kafka/Spark→Opensearch로 업로드하는 Docker 기반 스트리밍 E2E 파이프라인입니다.  
- **기술**: Docker Compose, Spark Structured Streaming, Kafka(kRaft), Opensearch, FastAPI/Nginx, Kiwipiepy(한국어 전처리)  
- **특징**: 마운트 폴더/파일 기반 스트리밍 수집 및 Spark 처리, Kafka/Opensearch로의 실시간 적재, 컨테이너별 네트워크 분리 및 검색용 웹 UI 연동

## B_project
- **설명**: 알고리즘 연습과 코딩 테스트 자동화에 쓰이는 알고리즘 라이브러리 및 유틸 모음입니다.  
- **기술**: Python 기반 알고리즘 구현(그래프/트리/문자열/고급 알고리즘), Jupyter 노트북, LLM 헬퍼 유틸리티  
- **특징**: boj_bible(기초·그래프·트리·문자열 등) 모듈 모음, CT 자동화 도구(LLM 호출 유틸 포함) 및 예제 노트북·운영 가이드 문서 추가
