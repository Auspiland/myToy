## geo_map
- **설명**: 사각형 영역 또는 좌표를 대한민국(남·북한 포함) 행정구역으로 변환하고 GeoJSON으로 시각화하는 도구입니다.  
- **기술**: Python, Shapely/STRtree 기반 공간색인, GeoJSON, GeoPandas 등 공간 처리 스택.  
- **특징**: 사각형→행정구역 변환(빠른 v2), 좌표→행정구역 변환 및 섬 자동 감지, GeoJSON 생성·배포(경계 조회 포함).

## Mini-Pipeline
- **설명**: YouTube 스크립트 추출부터 문장 전처리·Kafka·OpenSearch 업로드까지의 E2E 스트리밍 파이프라인입니다.  
- **기술**: Docker Compose(다중 네트워크), Spark Structured Streaming, Kafka(kRaft), OpenSearch, FastAPI + Nginx, Kiwipiepy(한국어 전처리).  
- **특징**: 컨테이너 기반 스트리밍 처리(파일→Kafka→Opensearch), Spark로 실시간 전처리(문장 분리·오탈자 보정), WEB 검색 인터페이스 제공.

## B_project
- **설명**: 알고리즘 연습 및 코딩 테스트 자동화·도구 모음 프로젝트입니다.  
- **기술**: Python 중심(알고리즘 라이브러리, Jupyter 노트북), FastAPI/Async 관련 도구 문서(운영 가이드 포함).  
- **특징**: boj_bible 알고리즘 모듈(자료구조·그래프·트리 등), CT 자동화·LLM 유틸(기본 모델 변경 안내 포함), 문제 풀이용 노트북/유틸 추가.
