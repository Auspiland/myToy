## [geo_map]
- **설명**: 사각형 영역 또는 좌표를 대한민국/북한의 행정구역(광역시·도·시군구 등)으로 변환하고 GeoJSON으로 출력하는 도구입니다.
- **기술**: Python, Shapely/GEOS (STRtree spatial index), GeoJSON, pyproj 등 지리공간 처리 스택.
- **특징**: 사각형→행정구역 변환 및 다수 일괄 처리, 좌표→행정구역 매핑·경계 조회, GeoJSON 생성·시각화(섬 자동 감지 포함).

## [Mini-Pipeline]
- **설명**: YouTube 스크립트 추출부터 전처리(Korean NLP)·Kafka·OpenSearch 업로드까지의 컨테이너화된 스트리밍 E2E 파이프라인입니다.
- **기술**: Docker(다중 compose), Spark Structured Streaming, Kafka(kRaft), OpenSearch, FastAPI/Nginx, Selenium, Kiwipiepy.
- **특징**: 실시간/스트리밍 데이터 흐름(Spark ↔ Kafka ↔ OpenSearch), 한국어 전처리(Kiwipiepy·룰베이스) 및 파이프라인 자동화, 웹 검색 인터페이스 연동(Nginx→FastAPI).

## [B_project]
- **설명**: 알고리즘 문제 풀이 및 코딩 테스트용 라이브러리와 자동화 도구들을 모아놓은 프로젝트입니다.
- **기술**: Python 기반 알고리즘 구현(모듈화된 라이브러리), Jupyter 노트북 예제, LLM 호출 유틸리티 포함.
- **특징**: boj_bible(자료구조·그래프·트리·문자열·고급 알고리즘) 모듈, CT(코딩 테스트) 자동화·LLM 헬퍼, 예제 노트북(kakao_history.ipynb) 및 운영·배포 가이드 문서.
