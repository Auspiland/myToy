## geo_map
- **설명**: 사각형 영역 또는 단일 좌표를 대한민국(남북)의 행정구역으로 빠르게 변환하고 GeoJSON으로 생성/시각화하는 도구입니다.  
- **기술**: Python, Shapely/STRtree 기반 공간 인덱스, GeoJSON, GeoPandas 기반 데이터 처리.  
- **특징**: 고성능 사각형→행정구역 변환(rect2region_fast_v2), 섬 자동 감지 및 방향 기반 대표 표현 생성, 행정구역 경계 조회 및 GeoJSON 파일 생성/일괄 변환.

## Mini-Pipeline
- **설명**: YouTube 스크립트 추출부터 전처리·Kafka 전송·Opensearch 적재까지의 컨테이너화된 스트리밍 E2E 파이프라인입니다.  
- **기술**: Docker / docker-compose, Spark Structured Streaming, Kafka(kRaft), OpenSearch, FastAPI, Nginx.  
- **특징**: 마운트 폴더 감시 기반 입력 → Spark→Kafka→OpenSearch 스트리밍 처리, Kiwipiepy 기반 문장 전처리(pandas_udf 사용), 컨테이너 네트워크 설계로 서비스 분리 및 확장성 확보.

## B_project
- **설명**: 알고리즘 연습·코딩테스트용 라이브러리와 자동화 도구들을 모은 프로젝트입니다.  
- **기술**: Python 기반 알고리즘 모듈 및 Jupyter 노트북, (LLM) 유틸리티 래퍼 포함.  
- **특징**: boj_bible(자료구조·그래프·트리 등 알고리즘 모듈), CT 자동화 도구 및 LLM 응답 유틸(response_GPT 계열) 제공, kakao_history.ipynb 등 문제 풀이 예제/노트북 포함.
