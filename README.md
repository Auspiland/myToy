## geo_map
- **설명**: 좌표 또는 사각형 영역을 대한민국/북한 행정구역으로 변환하고 GeoJSON으로 생성·시각화하는 공간 변환 도구입니다.  
- **기술**: Python, Shapely/GeoPandas 기반 공간 색인(STRtree), GeoJSON 처리.  
- **특징**: rect2region_fast_v2를 통한 고속 사각형→행정구역 변환, 섬 자동 감지 및 방향·포함률 기반 대표 표현, 행정구역 경계 조회 및 GeoJSON 생성/시각화.

## Mini-Pipeline
- **설명**: YouTube URL로부터 스크립트를 추출해 문장 전처리 후 Kafka·OpenSearch로 업로드하는 도커 기반 스트리밍 E2E 파이프라인입니다.  
- **기술**: Docker, Apache Spark(Structured Streaming), Apache Kafka, OpenSearch, FastAPI, Nginx, Kiwipiepy.  
- **특징**: 실시간 스트리밍 처리(파일 감시 → Kafka 토픽), Kiwipiepy·룰 기반 전처리(오탈자·문장분리 등), 웹 검색 인터페이스와 컨테이너화된 배포 구성.

## B_project
- **설명**: 알고리즘 연습 및 코딩 테스트 자동화를 위한 라이브러리·예제 모음입니다.  
- **기술**: Python, Jupyter Notebook 기반 예제·유틸, 알고리즘 구현(자료구조·그래프·트리·문자열 등), LLM 유틸 통합.  
- **특징**: boj_bible(기초·그래프·트리·문자열·고급) 알고리즘 모듈, CT 자동화·기록 도구 및 예제 노트북(kakao_history), LLM 응답 유틸(response_GPT 등) 관련 설정 변경 안내.
