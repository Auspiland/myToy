## geo_map
- **설명**: 좌표 또는 사각형 영역을 한국 행정구역(시/도/시군구 등)으로 변환하고 GeoJSON으로 시각화하는 파이썬 기반 툴입니다.  
- **기술**: Python, Shapely/STRtree 기반 공간 색인, GeoJSON, 공간 데이터(GeoJSON) 처리 도구들.  
- **특징**: 사각형/좌표→행정구역 변환(대량 처리 지원), 섬 자동 감지 및 방향 기반 대표 표현, 경계 정보 조회·GeoJSON 생성 및 배포 기능.

## Mini-Pipeline
- **설명**: YouTube 스크립트 추출부터 전처리·Kafka·Opensearch 업로드까지 Docker로 구성한 스트리밍 E2E 파이프라인입니다.  
- **기술**: Docker Compose, Spark Structured Streaming, Kafka(kRaft), Opensearch, FastAPI/Nginx, Selenium, Kiwipiepy.  
- **특징**: Spark ↔ Kafka ↔ Opensearch 흐름의 실시간 ETL, 컨테이너 네트워크 분리·오케스트레이션, 한국어 전처리(Kiwipiepy) 및 pandas_udf 기반 처리 최적화.

## B_project
- **설명**: 알고리즘/코딩테스트 연습용 라이브러리와 코딩 테스트 자동화 도구 모음입니다.  
- **기술**: Python(알고리즘 모듈·노트북), Jupyter, CT 자동화 유틸(LLM 호출 유틸 포함), FastAPI/DB/Redis 가이드 문서(운영 참조).  
- **특징**: boj_bible(자료구조·그래프·트리·문자열·고급 알고리즘) 모듈, CT 자동화·LLM 통합 유틸(모델 기본 파라미터 변경 기록 포함), kakao_history 노트북 등 예제·문서 제공.
