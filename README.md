## geo_map
- **설명**: 사각형 또는 점 좌표를 대한민국(남한/북한) 행정구역으로 변환하고 GeoJSON으로 시각화하는 도구.  
- **기술**: Python, Shapely/STRtree 기반 공간색인, GeoJSON, geopandas 스타일 데이터 처리.  
- **특징**: 사각형→행정구역 변환(단일/대량 처리), 좌표→지역 매핑 및 섬 자동 감지, 경계 조회·GeoJSON 생성 및 배포 지원.

## Mini-Pipeline
- **설명**: YouTube 스크립트 추출부터 전처리·Kafka 전송·Opensearch 적재까지의 스트리밍 E2E 파이프라인.  
- **기술**: Docker(Compose), Spark Structured Streaming, Kafka(kRaft), Opensearch, FastAPI, Nginx, Selenium, Kiwipiepy.  
- **특징**: 컨테이너 기반 네트워크 아키텍처, Spark readStream→Kafka→Opensearch 연동, Kiwipiepy 기반 문장 전처리(오탈자·토큰화 등).
