## geo_map
- **설명**: 한반도(남·북) 내 좌표나 사각형 영역을 행정구역(시/도/시군구 등)으로 변환하고 GeoJSON 생성 및 시각화를 지원합니다.  
- **기술**: Python 기반, GeoJSON, Shapely/STRtree 등 공간 인덱싱 및 경계 데이터 처리 도구.  
- **특징**: 사각형→행정구역/좌표→행정구역 변환, 섬 자동 감지(예: 제주 남서부), GeoJSON 생성·일괄 변환·경계 조회.

## Mini-Pipeline
- **설명**: 유튜브 스크립트 추출부터 문장 전처리, Kafka·OpenSearch 업로드까지 Docker 컨테이너 기반 스트리밍 E2E 파이프라인을 제공합니다.  
- **기술**: Docker Compose, Spark Structured Streaming, Kafka(kRaft), OpenSearch, FastAPI, Nginx.  
- **특징**: 스트리밍 기반 데이터 플로우(Kafka↔Spark↔OpenSearch), 한국어 전처리(Kiwipiepy + 룰베이스) 및 컨테이너화된 배포/네트워크 구성.

## B_project

- **기술**: Python 중심(알고리즘 모듈·노트북), FastAPI/Async·도커 관련 문서(운영 가이드 포함).
