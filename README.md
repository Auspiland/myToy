## geo_map
- **설명**: 좌표나 사각형 영역을 대한민국(남한/북한) 행정구역(시·도 등)으로 변환하고 GeoJSON으로 출력하는 도구입니다.
- **기술**: Python, Shapely/GeoPandas, STRtree 공간색인, GeoJSON
- **특징**: 사각형→행정구역 변환 및 대량 변환 지원(convert_many), 좌표→행정구역 변환 및 경계 조회(get_region_boundary), 섬 자동 감지·GeoJSON 생성 및 시각화

## Mini-Pipeline
- **설명**: YouTube 스크립트를 수집해 문장 전처리 후 Kafka와 OpenSearch로 스트리밍하여 검색·서비스까지 연결하는 E2E 파이프라인입니다.
- **기술**: Docker(컨테이너화), Apache Spark 스트리밍, Kafka, OpenSearch, FastAPI + Nginx
- **특징**: 컨테이너 기반 실시간 스트리밍 파이프라인, Kiwipiepy·룰베이스 기반 텍스트 전처리(F.pandas_udf 활용), OpenSearch 색인 및 웹 검색 인터페이스 연동

## B_project
- **설명**: 알고리즘 학습·코딩테스트 지원 라이브러리와 자동화 도구 모음입니다.
- **기술**: Python(모듈화된 알고리즘 패키지), Jupyter 노트북, FastAPI 관련 운영 문서
- **특징**: boj_bible(자료구조·그래프·문자열 등 알고리즘 모듈), CT(코딩테스트 자동화 및 기록 도구), LLM 유틸(기본 모델을 gpt-5-nano로 변경하는 헬퍼 함수 등)
