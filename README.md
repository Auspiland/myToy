## geo_map
- **설명**: 좌표나 사각형 영역을 대한민국/북한 행정구역으로 빠르게 변환하고 GeoJSON으로 시각화하는 도구  
- **기술**: Python, geopandas/shapely, STRtree 공간 인덱스, GeoJSON  
- **특징**: 사각형·좌표 → 행정구역 변환(고속화된 rect2region_fast_v2), 섬 자동 감지 및 대표 표현 생성, 행정구역 경계 조회·GeoJSON 일괄 생성·시각화

## Mini-Pipeline
- **설명**: YouTube 스크립트 수집부터 전처리·Kafka/Opensearch 업로드까지의 E2E 스트리밍 파이프라인  
- **기술**: Docker Compose, Spark Structured Streaming, Kafka(kRaft), Opensearch, FastAPI/Nginx, Kiwipiepy  
- **특징**: Spark→Kafka→Opensearch의 스트리밍 처리 파이프라인, Kiwipiepy+판다스 UDF 기반 한국어 전처리(오탈자·문장분리·불용어 처리), 컨테이너화된 서비스(검색 UI/관리) 구성

## B_project
- **설명**: 알고리즘 연습·코딩 테스트 지원용 라이브러리 및 도구 모음  
- **기술**: Python 기반 알고리즘 모듈, Jupyter 노트북, LLM 유틸 함수  
- **특징**: boj_bible(자료구조·그래프·트리·문자열·고급 알고리즘) 모듈, CT 자동화 및 문제 풀이 예제(노트북), LLM 연동 유틸(기본 모델이 gpt-5-nano로 변경된 호출 유틸 제공)
