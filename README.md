## geo_map
- **설명**: 사각형 영역 또는 좌표를 대한민국(남한/북한) 행정구역 명칭으로 변환하고 GeoJSON으로 시각화하는 도구입니다.  
- **기술**: Python, Shapely/GeoPandas, STRtree 기반 Spatial Index, GeoJSON.  
- **특징**: 사각형→행정구역 변환(대표 텍스트/상세 정보), 섬 지역 자동 감지 및 방향 기반 대표 표현, GeoJSON 생성 및 대량(JSON) 변환/경계 조회.

## Mini-Pipeline
- **설명**: YouTube 스크립트 추출부터 전처리, Kafka·Spark 스트리밍 파이프라인을 거쳐 OpenSearch에 색인하는 E2E 컨테이너화 파이프라인입니다.  
- **기술**: Docker Compose, Apache Spark(Structured Streaming), Kafka(kRaft), OpenSearch, FastAPI, Nginx, Kiwipiepy.  
- **특징**: 스트리밍 기반 데이터 흐름(Kafka↔Spark↔OpenSearch), 텍스트 전처리(Kiwipiepy·pandas_udf) 및 웹 검색 인터페이스, 멀티 네트워크/컨테이너 오케스트레이션 구성.

## B_project
- **설명**: 알고리즘 문제 풀이·코딩 테스트 지원 라이브러리와 관련 도구 모음입니다.  
- **기술**: Python 기반 알고리즘 모듈(백준용 라이브러리), Jupyter 노트북, FastAPI 및 운영·배포 문서.  
- **특징**: boj_bible(자료구조·그래프·트리·문자열 등) 모듈, CT(코딩 테스트) 자동화·LLM 헬퍼(기본 모델이 gpt-5 → gpt-5-nano로 변경) 및 예제 노트북·운영 가이드 추가.
