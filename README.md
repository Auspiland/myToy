## geo_map
- **설명**: 사각형 영역 또는 좌표를 대한민국(남한/북한) 행정구역으로 변환하고 경계/GeoJSON을 생성·시각화하는 도구입니다.
- **기술**: Python, Shapely/GEOS (STRtree), GeoJSON, 공간 인덱스 및 표준 GIS 데이터.
- **특징**: 사각형/포인트 → 행정구역 변환(대표 문구 제공), 섬 자동 감지 및 대표 표현 규칙, 행정구역 경계 조회 및 GeoJSON 파일 생성·시각화.

## Mini-Pipeline
- **설명**: YouTube 스크립트 추출부터 전처리·Kafka 전송·Spark 처리·OpenSearch 적재·웹 검색까지의 스트리밍 E2E 파이프라인입니다.
- **기술**: Docker(Compose), Spark Structured Streaming, Kafka(kRaft), OpenSearch, FastAPI/Nginx, Selenium/Kiwipiepy(텍스트 전처리).
- **특징**: 마운트 폴더 기반 스트리밍 ingest → Kafka 토픽 연동, Spark에서 Kiwipiepy 기반 전처리(pandas_udf) 및 Opensearch 적재, Docker 네트워크 구성으로 서비스 분리 및 웹 인터페이스 제공.

## B_project
- **설명**: 알고리즘 연습 및 코딩 테스트 도구 모음(라이브러리, 자동화 스크립트, 참고 노트북 포함)입니다.
- **기술**: Python, Jupyter 노트북, 알고리즘 구현(그래프/트리/문자열/고급 알고리즘).
- **특징**: boj_bible(자료구조·알고리즘 라이브러리), CT(코딩 테스트 자동화 및 LLM 호출 유틸 — 기본 모델 변경 안내 포함), 문제 풀이·분석용 노트북 및 운영·개발 가이드 문서.
