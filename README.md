## geo_map
- **설명**: 좌표나 사각형 영역을 한국(남·북한) 행정구역으로 매핑하고 GeoJSON으로 시각화하는 도구입니다.
- **기술**: Python, Shapely/STRtree(Spatial Index), GeoJSON/Geopandas 기반 처리
- **특징**: 사각형·점 → 행정구역 변환(대표 표현 포함), 행정구역 경계 조회 및 GeoJSON 생성·시각화, 대량 변환·고성능 처리(배치 변환 + 섬 자동 감지)

## Mini-Pipeline
- **설명**: YouTube URL로부터 스크립트를 추출해 문장 전처리 후 Kafka와 OpenSearch에 업로드하는 Docker화된 스트리밍 E2E 파이프라인입니다.
- **기술**: Docker(컨테이너·docker‑compose), Spark (Structured Streaming), Kafka, OpenSearch, FastAPI, Nginx, Kiwipiepy, Selenium
- **특징**: Spark readStream → Kafka 토픽 기반 스트리밍 파이프라인, Kiwipiepy·룰베이스 전처리 및 Opensearch 색인, 컨테이너 네트워크 설계(Nginx 라우팅·서비스 분리)

## B_project
- **설명**: 알고리즘 학습·코딩테스트 지원용 리포지토리로, 문제 풀이 라이브러리와 자동화 도구를 포함합니다.
- **기술**: Python, Jupyter Notebook, 알고리즘 구현(자료구조·그래프·트리 등), LLM 호출 유틸리티
- **특징**: boj_bible(기본 자료구조·그래프·문자열·고급 알고리즘) 모듈, CT 자동화·기록 도구(LLM helper 함수 포함, 기본 모델 변경 안내), kakao_history.ipynb 등 실전 예제 노트북 포함
