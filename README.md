## geo_map
- **설명**: 좌표 또는 사각형 영역을 대한민국(남한/북한) 행정구역으로 변환하고 GeoJSON으로 생성·시각화합니다.
- **기술**: Python, Shapely/GeoPandas 기반 공간처리, STRtree 공간 인덱스, GeoJSON.
- **특징**: 사각형→행정구역 변환(대표 텍스트/상세 정보), 대량 처리 성능 최적화 및 섬 자동 감지, 경계 조회·GeoJSON 생성·시각화 기능.

## Mini-Pipeline
- **설명**: YouTube URL로부터 스크립트 추출→문장 전처리→Kafka/Opensearch 업로드까지의 도커화된 스트리밍 E2E 파이프라인입니다.
- **기술**: Docker Compose, Apache Spark(Structured Streaming), Kafka(kRaft), OpenSearch, FastAPI/Nginx, Kiwipiepy(한국어 전처리).
- **특징**: Spark↔Kafka↔OpenSearch 스트리밍 연동, 마이크로서비스 도커 네트워크 구성(Nginx→FastAPI), 한국어 전처리(pandas_udf·Kiwi) 및 파일 감시 기반 입력 파이프라인.

## B_project
- **설명**: 알고리즘 및 코딩 테스트 풀이용 라이브러리·도구 모음과 실습용 노트북/문서입니다.
- **기술**: Python 기반 알고리즘 모듈 및 Jupyter 노트북, CT 자동화 스크립트, LLM 유틸(응답 함수).
- **특징**: boj_bible(자료구조·그래프·트리·문자열·고급 알고리즘) 모듈, CT 자동화·기록 도구 및 kakao_history.ipynb 예제, LLM 헬퍼 함수(response_GPT 등) 기본 모델 변경 안내.
