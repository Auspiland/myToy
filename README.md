## geo_map
- **설명**: 사각형 영역 또는 좌표를 한국(남한/북한) 행정구역으로 변환하고 GeoJSON을 생성·시각화하는 도구  
- **기술**: Python, Shapely/STRtree 기반 공간 인덱싱, GeoJSON  
- **특징**: 사각형→행정구역 변환(광역/기초단위), 섬 지역 자동 감지 및 경계 조회, GeoJSON 생성 및 일괄 변환

## Mini-Pipeline
- **설명**: YouTube 스크립트 추출 후 문장 전처리하여 Kafka·Opensearch로 스트리밍 저장하는 E2E 파이프라인  
- **기술**: Docker(Compose), Spark Structured Streaming, Kafka(kRaft), Opensearch, FastAPI/Nginx  
- **특징**: 파일 감시→Spark→Kafka→Opensearch 스트리밍 워크플로우, 한국어 전처리(Kiwipiepy+룰 기반), 컨테이너화된 네트워크 구성

## B_project

- **기술**: Python 기반 알고리즘 모듈, FastAPI/Async DB(문서 참고)  
- **특징**: boj_bible(자료구조·그래프·문자열 등) 모듈, CT 자동화 유틸(LLM 연동 관련 기본 모델 변경 안내 포함), 문제 풀이 예제·노트북 포함
