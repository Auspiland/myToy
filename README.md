## geo_map
- **설명**: 사각형 영역 또는 좌표를 대한민국(남한/북한) 행정구역으로 변환하고 GeoJSON으로 시각화하는 도구입니다.  
- **기술**: Python, Shapely/STRtree 기반 공간 색인, GeoJSON, pyproj 등 GIS 스택.  
- **특징**: 사각형/좌표 → 행정구역 변환(광역/시군구 단위), 섬 자동 감지 및 방향 기반 대표표현 생성, GeoJSON 생성·배포·일괄 변환 지원.

## Mini-Pipeline
- **설명**: Youtube 스크립트 추출부터 문장 전처리, Kafka·Opensearch 업로드까지 동작하는 컨테이너 기반 스트리밍 E2E 파이프라인입니다.  
- **기술**: Docker Compose, Spark Structured Streaming, Kafka(kRaft), OpenSearch, FastAPI/Nginx, Kiwipiepy(전처리).  
- **특징**: 컨테이너 네트워크 기반 모듈화(웹/API/스파크/검색), Spark로 파일→Kafka→OpenSearch 스트리밍 처리, Kiwipiepy+룰베이스 기반 텍스트 정제 및 배치/실시간 전처리.

## B_project
- **설명**: 알고리즘 학습 및 코딩테스트 보조용 라이브러리와 자동화 도구 모음입니다.  
- **기술**: Python(모듈화된 알고리즘 패키지), Jupyter 노트북, FastAPI/LLM 유틸(보조문서).  
- **특징**: boj_bible(자료구조·그래프·트리·문자열·고급 알고리즘) 구성, CT 자동화 및 LLM 호출 유틸(기본 모델 변경 안내 포함), kakao_history.ipynb 등 문제 풀이 예제 제공.
