## [geo_map]
- **설명**: 좌표 또는 사각형 영역을 대한민국(남한/북한) 행정구역으로 빠르게 변환하고 시각화하는 도구입니다.
- **기술**: Python, Shapely/GeoPandas, STRtree 공간인덱스, GeoJSON
- **특징**: 사각형→행정구역 고속 변환 및 다중 처리, 섬 자동 감지·방향 기반 대표 표현 생성, 경계 조회 및 GeoJSON 생성/시각화

## [Mini-Pipeline]
- **설명**: YouTube 스크립트 수집부터 전처리·Kafka 전송·OpenSearch 적재까지의 E2E 스트리밍 파이프라인입니다.
- **기술**: Docker/Docker Compose, Spark Structured Streaming, Kafka(kRaft), OpenSearch, FastAPI, Nginx
- **특징**: 컨테이너화된 스트리밍 아키텍처, Spark→Kafka→OpenSearch 데이터 흐름, Kiwipiepy 기반 문장 전처리(오탈자 수정·문장 분리 등)
