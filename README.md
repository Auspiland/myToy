## geo_map
- **설명**: 좌표 또는 사각형 영역을 대한민국/북한 행정구역(시/도 등)으로 변환해 결과를 분석·시각화하는 도구  
- **기술**: Python, Shapely/GeoPandas 기반 공간색인(STRtree), GeoJSON 생성/처리  
- **특징**: 사각형·좌표→행정구역 변환 및 배치 처리, 섬 자동 감지(지역 방향/포함률 규칙), GeoJSON 생성 및 경계/중심점 조회

## Mini-Pipeline
- **설명**: Youtube 스크립트 추출부터 문장 전처리·Kafka·Opensearch 업로드까지의 E2E 스트리밍 파이프라인(도커 기반)  
- **기술**: Docker + docker-compose, Spark Structured Streaming, Kafka(kRaft), OpenSearch, FastAPI/Nginx, Kiwipiepy(형태소 처리)  
- **특징**: 마운트 폴더 감시→Spark로 Kafka/Opensearch 저장(Streaming), Kiwi 기반 전처리 및 pandas_udf 적용, 다중 컨테이너 네트워크 구성으로 서비스 분리

## B_project
- **설명**: 알고리즘 문제 풀이·코딩 테스트 보조 라이브러리 및 관련 도구 모음  
- **기술**: Python 기반 알고리즘 모듈(자료구조·그래프·트리·문자열 등), Jupyter 노트북, FastAPI/Async 관련 운영 가이드 문서  
- **특징**: boj_bible 알고리즘 라이브러리, CT 자동화(LLM 헬퍼 함수 포함 — 기본 모델 변경 알림), 문제 풀이용 노트북 및 실습 예제 추가
