## geo_map
- **설명**: 사각형 영역 또는 좌표를 대한민국(남한/북한) 행정구역(시/도/시군구 등)으로 변환하고 GeoJSON으로 시각화하는 도구입니다.
- **기술**: Python, Shapely/STRtree, GeoJSON, GeoPandas(데이터 전처리), GitHub 연동 스크립트
- **특징**: 사각형→행정구역 변환(대량 처리 지원), 섬 자동 감지 및 대표 텍스트 생성, 행정구역 경계 조회 및 GeoJSON 생성/시각화

## Mini-Pipeline
- **설명**: YouTube 스크립트 추출부터 문장 전처리, Kafka·OpenSearch 업로드까지의 스트리밍 E2E 파이프라인입니다.
- **기술**: Docker Compose, Apache Spark(Structured Streaming), Kafka(kRaft), OpenSearch, FastAPI + Nginx, Selenium(스크립트 수집)
- **특징**: 컨테이너 기반 스트리밍 처리 파이프라인, Kiwipiepy 기반 텍스트 전처리(pandas_udf 적용), Kafka 토픽 중심의 데이터 흐름 및 OpenSearch 인덱싱/검색

## B_project
- **설명**: 알고리즘 문제 풀이·코딩 테스트 연습을 위한 유틸리티와 예제 모음입니다.
- **기술**: Python, Jupyter Notebook, 알고리즘 구현(그래프/트리/문자열/고급 알고리즘), LLM 연동 유틸
- **특징**: boj_bible(자료구조·알고리즘 라이브러리), CT 자동화 도구 및 LLM 호출 유틸(기본 모델 변경 안내 포함), 문제 풀이용 노트북과 예제 코드 제공
