## geo_map
- **설명**: 사각형 영역 또는 좌표를 대한민국(남한/북한) 행정구역으로 변환하고 GeoJSON으로 시각화하는 도구  
- **기술**: Python, geopandas/shapely, STRtree 공간색인, GeoJSON (행정구역 셰이프/데이터)  
- **특징**: 사각형→시/도 단위 변환 및 다수 처리(배치), 좌표→행정구역 매핑(포인트 변환), 섬 자동 감지·경계 조회 및 GeoJSON 생성

## Mini-Pipeline
- **설명**: 유튜브 스크립트를 수집해 전처리하고 Kafka·Opensearch에 업로드하는 컨테이너 기반 스트리밍 파이프라인  
- **기술**: Docker/Docker Compose, Apache Spark Structured Streaming, Kafka(kRaft), OpenSearch, FastAPI + Nginx, Kiwipiepy (한국어 전처리)  
- **특징**: Spark 스트리밍을 통한 E2E 처리(Kafka 토픽 입출력 → OpenSearch 색인), 컨테이너화된 마이크로서비스 아키텍처, 한국어 전처리(토큰화·룰베이스 오탈자 보정)

## B_project
- **설명**: 알고리즘 문제 풀이(코딩 테스트) 라이브러리 및 자동화 도구 모음  
- **기술**: Python 기반 알고리즘/유틸 모듈, Jupyter 노트북 예제, LLM 연동 유틸(CT 자동화)  
- **특징**: boj_bible(자료구조·그래프·트리·문자열 등 알고리즘 모듈), CT 자동화(LLM 래퍼·스트리밍 지원) 및 예제 노트북·운영·배포 가이드 문서
