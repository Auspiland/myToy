## geo_map
- **설명**: 한반도 좌표나 사각형 영역을 시/도/시군구 단위 행정구역으로 변환하고 GeoJSON으로 출력·시각화하는 도구입니다.
- **기술**: Python, Shapely/GeoPandas, pyproj, STRtree 공간색인, GeoJSON
- **특징**: 사각형→행정구역 대량 변환(convert_many) 및 좌표→행정구역 변환, 섬 자동 감지(섬 포함 비율 기반), 행정구역 경계 조회 및 GeoJSON 생성·배포 자동화

## Mini-Pipeline
- **설명**: YouTube 스크립트 추출부터 전처리·Kafka·Spark를 거쳐 Opensearch에 업로드하는 도커 기반 스트리밍 E2E 파이프라인입니다.
- **기술**: Docker Compose, Apache Kafka(kRaft), Spark Structured Streaming, FastAPI, Opensearch, Kiwipiepy
- **특징**: 컨테이너별 분리된 네트워크 아키텍처(Nginx 라우팅 포함), Spark+Kafka 기반 실시간 전처리·토픽 파이프라인, Opensearch 연동 및 웹 검색 인터페이스

## B_project
- **설명**: 알고리즘/코딩 테스트 풀이용 라이브러리와 자동화·참고 문서들을 모아둔 프로젝트입니다.
- **기술**: Python(알고리즘 구현, Jupyter), LLM 호출 유틸리티, (문서) FastAPI/Async/DB 관련 스택
- **특징**: boj_bible(자료구조·그래프·트리·문자열·고급 알고리즘) 모듈, CT 자동화 및 LLM 유틸(기본 모델 gpt-5-nano로 변경) 설명, 문제 풀이 예제 노트북 및 운영·개발 가이드 문서
