## geo_map
- **설명**: 좌표 또는 사각형 영역을 한국(남한/북한) 행정구역으로 변환하고 GeoJSON으로 생성·시각화하는 도구입니다.
- **기술**: Python, GeoPandas/Shapely, STRtree 공간인덱스, GeoJSON, 일부 LLM(지명 번역).
- **특징**: 사각형·좌표→행정구역 변환(대표문구/상세), 행정구역 경계 조회 및 GeoJSON 생성·시각화, STRtree 기반 고성능 처리 및 섬 자동 감지.

## Mini-Pipeline
- **설명**: YouTube URL에서 스크립트를 추출해 전처리하고 Kafka/Opensearch로 업로드하는 Docker 기반 스트리밍 파이프라인입니다.
- **기술**: Docker Compose, Spark Structured Streaming, Kafka(kRaft), Opensearch, FastAPI, Nginx, Selenium, Kiwipiepy.
- **특징**: 파일/스트림 감시 → Spark로 변환 후 Kafka 토픽/Opensearch에 저장, Kiwipiepy 기반 문장 전처리(오탈자·분할·불용어), 컨테이너화된 마이크로서비스 아키텍처 및 웹 검색 인터페이스.

## B_project
- **설명**: 알고리즘 학습·코딩테스트용 라이브러리와 자동화 도구를 모아놓은 프로젝트입니다.
- **기술**: Python 기반 알고리즘 코드 및 노트북(.ipynb), FastAPI/LLM 연동 유틸(CT 도구).
- **특징**: boj_bible(자료구조·그래프·트리·문자열·고급 알고리즘) 모듈, CT 자동화/기록 도구(LLM 헬퍼 함수 포함) 및 문제풀이 노트북, 운영·배포 가이드 문서(special_prompt.md).
