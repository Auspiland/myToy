## geo_map
- **설명**: 좌표 또는 사각형 영역을 대한민국/북한 행정구역(광역/기초)으로 변환하고 GeoJSON으로 시각화하는 툴  
- **기술**: Python, Shapely/GeoPandas, STRtree 공간인덱스, GeoJSON  
- **특징**: 사각형·좌표 → 행정구역 변환(대표문구/상세), 섬 자동 감지 및 대표 표현 규칙, 행정구역 경계 조회·GeoJSON 생성 및 일괄 변환

## Mini-Pipeline
- **설명**: YouTube 스크립트 추출부터 문장 전처리·Kafka/Opensearch 업로드까지의 스트리밍 E2E 파이프라인  
- **기술**: Docker Compose(멀티 컨테이너), Spark Structured Streaming, Kafka, Opensearch, FastAPI, Nginx, Kiwipiepy  
- **특징**: Streaming 기반 데이터 흐름(파일 감시 → Spark → Kafka → Opensearch), Kiwi 기반 문장 전처리(pandas_udf 활용), 컨테이너화된 서비스 구성 및 웹 검색 인터페이스 연동

## B_project
- **설명**: 알고리즘 문제 풀이와 코딩 테스트 자동화·보조 도구 모음 프로젝트  
- **기술**: Python(알고리즘 라이브러리 및 유틸), Jupyter Notebook, FastAPI/Async 관련 스크립트(문서)  
- **특징**: boj_bible(자료구조·알고리즘 모듈) 제공, CT(코딩 테스트 자동화) 및 LLM 헬퍼(response_GPT 등) 포함, 문제 풀이 노트북·운영·프롬프트 문서 제공
