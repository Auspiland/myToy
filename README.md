## geo_map
- **설명**: 사각형 영역 또는 좌표를 대한민국(남한/북한) 행정구역으로 빠르게 변환하고 시각화하는 도구입니다.  
- **기술**: Python 기반(Shapely/GeoPandas 기반 공간 인덱스 STRtree, GeoJSON), GeoJSON 데이터 사용.  
- **특징**: 사각형→행정구역/좌표→행정구역 변환(대량 처리 지원), 행정구역 경계 조회·섬 자동 감지, GeoJSON 생성 및 시각화/파일 일괄 변환.

## Mini-Pipeline
- **설명**: 유튜브 URL로부터 스크립트를 추출해 문장 전처리 후 Kafka와 OpenSearch로 스트리밍 업로드하는 E2E 파이프라인입니다.  
- **기술**: Docker 컴포즈 기반 컨테이너화, Apache Spark Structured Streaming, Kafka(kRaft), OpenSearch, FastAPI/Nginx.  
- **특징**: 스트리밍 인gestion(파일/selenium 마운트), Spark에서 Kiwipiepy 기반 한글 전처리·룰베이스 처리 및 Kafka 토픽/Opensearch 색인, 웹 검색 인터페이스 제공.

## B_project
- **설명**: 알고리즘 연습과 코딩 테스트 해결을 위한 파이썬 라이브러리 및 도구 모음입니다.  
- **기술**: Python (알고리즘 모듈, Jupyter 노트북), 유틸 스크립트 및 문서(LLM 래퍼 포함).  
- **특징**: boj_bible(자료구조·그래프·트리·문자열·고급 알고리즘) 모듈, CT 자동화·기록 도구(LLM 호출 유틸 변경 기록 포함), 문제 풀이 예제 노트북(예: kakao_history.ipynb).
