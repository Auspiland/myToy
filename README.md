## geo_map
- **설명**: 사각형 영역 또는 좌표를 대한민국(남·북한 포함) 행정구역명으로 변환하고 GeoJSON으로 시각화하는 도구입니다.
- **기술**: Python, Shapely/STRtree 기반 공간색인, GeoJSON, GitHub 연동(파일 생성/푸시), LLM(지명 번역) 보조.
- **특징**: 사각형→행정구역 변환(대/시/군 단위), 섬 자동 감지 및 방향 기반 대표표현 생성, 경계 조회 및 GeoJSON 일괄 생성/변환 기능.

## Mini-Pipeline
- **설명**: YouTube 스크립트 추출부터 문장 전처리·Kafka·Opensearch 업로드까지의 스트리밍 E2E 파이프라인입니다.
- **기술**: Docker Compose, Spark Structured Streaming, Kafka(kRaft), Opensearch, FastAPI/Nginx, Selenium(스크립트 수집), Kiwipiepy(한국어 전처리).
- **특징**: 컨테이너 기반 네트워크 아키텍처(서비스 분리), Spark로 파일·Kafka 스트리밍 처리 및 pandas_udf 활용한 NLP 전처리, Opensearch 연동 및 웹 검색 제공.

## B_project
- **설명**: 알고리즘 문제 풀이용 라이브러리와 코딩 테스트 자동화/기록 도구 모음입니다.
- **기술**: Python(알고리즘 모듈, 노트북), FastAPI/Async DB 관련 문서(운영 가이드), LLM 호출 유틸리티 포함.
- **특징**: boj_bible(자료구조·알고리즘 모듈) 구성, CT 자동화 도구(LLM 호출 유틸 변경사항 포함 — 기본 모델 gpt-5→gpt-5-nano), kakao_history 노트북 등 예제 코드 제공.
