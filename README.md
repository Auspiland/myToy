## geo_map
- **설명**: 사각형 영역이나 좌표를 대한민국(남한/북한) 행정구역으로 변환하고 GeoJSON으로 시각화하는 도구  
- **기술**: Python, Shapely/GeoPandas, STRtree 공간 인덱스, GeoJSON  
- **특징**: 사각형→행정구역 변환(광역/시·군·구), 좌표→행정구역 매핑, 섬 자동 감지 및 GeoJSON 생성/일괄 변환

## Mini-Pipeline
- **설명**: YouTube 스크립트 수집부터 전처리·Kafka·Opensearch 업로드까지의 E2E 스트리밍 파이프라인  
- **기술**: Docker-compose, Spark Structured Streaming, Kafka(kRaft), Opensearch, FastAPI, Nginx, Kiwipiepy  
- **특징**: 컨테이너 기반 스트리밍 아키텍처, Spark로 Kafka↔Opensearch 연동·실시간 처리, 한국어 전처리(키위·룰베이스) 및 검색 서비스 연동
