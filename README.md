## geo_map
- **설명**: 좌표 또는 사각형 영역을 대한민국(남한/북한) 행정구역(시/도/시군구 등)으로 변환하고 GeoJSON으로 시각화하는 도구  
- **기술**: Python, Shapely/STRtree 기반 공간 인덱스, GeoJSON 데이터 처리  
- **특징**: 고속 사각형→행정구역 변환(rect2region_fast_v2), 좌표→행정구역 매핑 및 경계 조회, 섬 지역 자동 감지 및 GeoJSON 생성/출력

## Mini-Pipeline
- **설명**: YouTube 스크립트 추출 → 문장 전처리 → Kafka/Spark 스트리밍 파이프라인 → OpenSearch 업로드를 처리하는 E2E 컨테이너화된 시스템  
- **기술**: Docker 기반 컨테이너, Apache Kafka, Apache Spark(Structured Streaming), OpenSearch, FastAPI/Nginx, Kiwipiepy(한국어 전처리)  
- **특징**: Streaming 기반 데이터 수집·처리 파이프라인, Spark로 Kafka 토픽 입출력 및 전처리(F.pandas_udf 활용), OpenSearch 연동을 통한 검색 UI 제공

## B_project
- **설명**: 알고리즘 학습·코딩 테스트 연습을 위한 문제 풀이 라이브러리 및 자동화 도구 모음  
- **기술**: Python 중심 구현, Jupyter 노트북 예제, 알고리즘 모듈화(그래프/트리/문자열/고급 알고리즘), LLM 유틸리티 포함  
- **특징**: boj_bible(자료구조·알고리즘 구현) 모듈, CT(코딩 테스트) 자동화 도구 및 예제 노트북 추가, LLM 호출 유틸(기본 모델 변경 등)
