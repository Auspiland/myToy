## geo_map
- **설명**: 입력한 좌표 또는 사각형 영역을 대한민국(남한/북한) 행정구역 단위로 빠르게 매핑하고 GeoJSON으로 시각화하는 도구입니다.
- **기술**: Python (Shapely/GeoPandas 기반), STRtree 공간 인덱스, GeoJSON, GitHub 연동 스크립트
- **특징**: 사각형→행정구역 및 좌표 변환(대량처리 지원), 행정구역 경계 조회·GeoJSON 생성·시각화, 섬 자동 감지 및 대표 텍스트 생성

## Mini-Pipeline
- **설명**: YouTube 스크립트 추출부터 문장 전처리·Kafka 전송·Spark 처리·Opensearch 업로드까지의 E2E 스트리밍 파이프라인입니다.
- **기술**: Docker Compose(컨테이너화), Spark Structured Streaming, Kafka(kRaft), Opensearch, FastAPI/Nginx, Selenium/Kiwipiepy
- **특징**: 컨테이너 기반의 스트리밍 아키텍처(네트워크 분리), Spark↔Kafka↔Opensearch 연동을 통한 실시간 처리·색인, 한국어 전처리(Kiwi) 및 검색 UI 연동

## B_project
- **설명**: 알고리즘 학습·코딩 테스트 실습용 레포지토리(문제 풀이 라이브러리·자동화 도구 포함)입니다.
- **기술**: Python 기반 알고리즘 모듈, Jupyter 노트북, FastAPI/데브 가이드 문서(운영·배포 관련 문서 포함)
- **특징**: boj_bible(자료구조·그래프·트리·문자열 등 알고리즘 모듈), CT(코딩 테스트 자동화·LLM 연동 유틸), 예제 노트북 및 개발/배포 가이드 문서
