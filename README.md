## geo_map
- **설명**: 사각형 영역 또는 좌표를 대한민국(남한/북한) 행정구역으로 변환하고 GeoJSON으로 시각화하는 툴입니다.  
- **기술**: Python, Shapely/STRtree, GeoPandas/GeoJSON, 공간 인덱스 및 GitHub 자동 푸시 스크립트.  
- **특징**: 사각형→행정구역 변환(배치 처리/성능 최적화), 좌표→행정구역 변환 및 경계 조회, 섬 자동 감지·GeoJSON 생성 및 시각화.

## Mini-Pipeline
- **설명**: YouTube 스크립트 추출 후 문장 전처리하여 Kafka와 OpenSearch에 업로드하는 E2E 스트리밍 파이프라인입니다.  
- **기술**: Docker(멀티 컨테이너), Apache Spark Structured Streaming, Kafka(kRaft), OpenSearch, FastAPI/Nginx, Selenium, Kiwipiepy.  
- **특징**: 스트리밍 기반 Spark→Kafka→OpenSearch 워크플로우, Kiwi 기반 한국어 전처리(문장 분리·오탈자 보정), Docker Compose로 네트워크·서비스 분리 운영.

## B_project
- **설명**: 알고리즘 학습·코딩테스트 도우미 라이브러리 및 자동화 도구 모음입니다.  
- **기술**: Python, Jupyter 노트북, 알고리즘 모듈(자료구조/그래프/트리/문자열 등), LLM 호출 유틸리티(FastAPI 연동 가능).  
- **특징**: boj_bible(자료구조·알고리즘 구현 모음), CT 자동화·기록 도구(LLM 기본 모델 변경 포함), 문제 풀이·샘플 함수가 담긴 kakao_history 노트북 제공.
