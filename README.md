## geo_map
- **설명**: 좌표 또는 사각형 영역을 대한민국(남한/북한) 행정구역 단위로 변환하고 GeoJSON으로 출력·시각화하는 도구입니다.  
- **기술**: Python, Shapely/GeoPandas, STRtree 공간인덱스, GeoJSON, LLM 보조 번역(지명).  
- **특징**: 사각형→행정구역 변환(대량 처리 지원), 좌표→행정구역 매칭·경계 조회, 섬 자동 감지 및 GeoJSON 생성/시각화.

## Mini-Pipeline
- **설명**: YouTube 스크립트 추출부터 전처리·Kafka·Opensearch 업로드까지의 스트리밍 E2E 파이프라인입니다.  
- **기술**: Docker(Compose), Apache Spark Structured Streaming, Kafka(kRaft), OpenSearch, FastAPI, Nginx, Kiwipiepy, Selenium(수집 보조).  
- **특징**: 컨테이너 기반 스트리밍 아키텍처, Spark→Kafka→Opensearch 연동으로 실시간 처리·검색, 한국어 전처리(오탈자·문장분리) 최적화.

## B_project
- **설명**: 알고리즘 학습·코딩테스트 풀이용 라이브러리 및 도구 모음입니다.  
- **기술**: Python 중심(모듈/노트북), 알고리즘·자료구조 구현, LLM 연동 유틸리티(CT 자동화에 사용).  
- **특징**: boj_bible(자료구조·그래프·트리·문자열 등) 모듈화, CT 자동화·LLM 기반 헬퍼 함수, 문제 풀이용 노트북·예제 코드 포함.
