## geo_map
- **설명**: 좌표 또는 사각형 영역을 대한민국·북한 행정구역 명칭으로 변환하고 GeoJSON으로 시각화하는 파이썬 도구입니다.
- **기술**: Python, Shapely/STRtree(Spatial Index), GeoJSON, pyproj 등 GIS 관련 라이브러리.
- **특징**: 사각형/좌표 → 행정구역 변환(대표표현 포함), 섬 자동 감지 및 방향 기반 설명 생성, GeoJSON 파일 생성·일괄 변환 및 경계 조회 기능.

## Mini-Pipeline
- **설명**: YouTube 스크립트를 수집해 문장 전처리 후 Kafka와 OpenSearch로 업로드하는 Docker 기반 스트리밍 E2E 파이프라인입니다.
- **기술**: Docker Compose, Spark Structured Streaming, Kafka(kRaft), OpenSearch, FastAPI, Nginx, Kiwipiepy, Selenium(수집 보조).
- **특징**: Spark→Kafka→OpenSearch로 이어지는 실시간 처리 파이프라인, Kiwipiepy 기반 전처리(pandas_udf) 및 룰베이스 보정, 멀티 컨테이너 네트워크 구성으로 서비스 분리 운영.

## B_project
- **설명**: 알고리즘 문제 풀이 및 코딩 테스트 자동화·도구 모음 프로젝트입니다.
- **기술**: Python 중심(자료구조·그래프·트리·문자열·고급 알고리즘 모듈), Jupyter 노트북 및 FastAPI/DB 운영 가이드 문서 참조.
- **특징**: boj_bible로 핵심 알고리즘/자료구조 제공, CT 자동화 도구(LLM 유틸 포함) 및 kakao_history 노트북에 포함된 예제 풀이, AI 포털 운영·배포 가이드(special_prompt.md).
