## geo_map
- **설명**: 좌표나 사각형 영역을 대한민국(및 북한) 행정구역으로 변환하고 GeoJSON으로 시각화하는 도구입니다.
- **기술**: Python, shapely/STRtree 공간 인덱스, GeoJSON/Geopandas 기반 경계 데이터.
- **특징**: 사각형·좌표 → 행정구역 변환; 섬 자동 감지 및 면적/방향 기반 대표 표현 규칙; GeoJSON 생성·경계 조회·대량 변환 지원.

## Mini-Pipeline
- **설명**: YouTube 스크립트 추출 후 문장 전처리하여 Kafka와 OpenSearch로 스트리밍 업로드하는 Docker화된 E2E 파이프라인입니다.
- **기술**: Docker, Apache Spark(Structured Streaming), Kafka(kRaft), OpenSearch, FastAPI/Nginx, Kiwipiepy(한국어 전처리).
- **특징**: Streaming 기반 입력→처리→저장 파이프라인; Spark↔Kafka↔OpenSearch 연동 및 웹 검색 인터페이스; Selenium과 마운트 방식으로 스크립트 수집 지원.

## B_project
- **설명**: 알고리즘 연습 및 코딩 테스트 자동화·보조 도구들을 모아둔 프로젝트입니다.
- **기술**: Python 기반 알고리즘 모듈·노트북, LLM 연동 유틸(응답 함수/스트리밍).
- **특징**: boj_bible(자료구조·알고리즘 라이브러리); CT 자동화 및 전처리 도구(예: Kiwipiepy 사용) 및 LLM 기본 모델 변경 안내(gpt-5 → gpt-5-nano); kakao_history.ipynb 등 예제 포함.
