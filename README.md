## geo_map
- **설명**: 사각형 영역 또는 좌표를 대한민국(남한/북한) 행정구역으로 변환하고 경계/GeoJSON을 생성하는 도구  
- **기술**: Python, GeoJSON, Shapely/STRtree(Spatial index), GeoJSON 데이터(행정구역), GitHub 연동  
- **특징**: 사각형→행정구역/좌표→행정구역 변환(대량 처리 지원), 섬 지역 자동 감지 및 대표 텍스트 생성, 경계 조회·GeoJSON 파일 생성 및 시각화

## Mini-Pipeline
- **설명**: YouTube 스크립트 추출부터 전처리·Kafka 전송·Opensearch 저장까지의 스트리밍 E2E 파이프라인  
- **기술**: Docker(Compose), Apache Spark Structured Streaming, Kafka(kRaft), Opensearch, FastAPI, Nginx, Kiwipiepy  
- **특징**: 컨테이너 기반 멀티서비스 아키텍처(Nginx→FastAPI), Spark를 통한 스트리밍 전처리(Kiwi + pandas_udf) 및 Kafka/Opensearch 연동, 웹 검색 인터페이스 제공

## B_project
- **설명**: 알고리즘/코딩테스트 풀이용 라이브러리·도구 모음 및 관련 실습 자료 저장소  
- **기술**: Python(알고리즘 구현), Jupyter Notebook, FastAPI/Async(운영 가이드 문서)  
- **특징**: boj_bible(자료구조·그래프·트리·문자열·고급 알고리즘) 모듈, CT 자동화·기록 도구(LLM 호출 유틸 포함) 및 기본 모델 변화(gpt-5 → gpt-5-nano) 안내, kakao_history 노트북의 문제 풀이 예제 포함
