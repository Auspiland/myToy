## geo_map
- **설명**: 좌표나 사각형 영역을 한국 행정구역(남한/북한) 단위로 변환하고 경계/GeoJSON을 생성하는 도구입니다.
- **기술**: Python, Shapely/GeoPandas(공간 인덱스 STRtree), GeoJSON, pyproj 등 지리공간 처리 스택.
- **특징**: 사각형→행정구역(광역/시도 포함) 변환 및 배치 처리, 섬 자동 감지·대표 텍스트 규칙, 경계 조회·GeoJSON 생성 및 시각화/일괄 변환.

## Mini-Pipeline
- **설명**: YouTube 스크립트 추출 → 문장 전처리 → Kafka/Opensearch 업로드로 이어지는 컨테이너화된 스트리밍 E2E 파이프라인입니다.
- **기술**: Docker Compose, Spark Structured Streaming, Kafka(kRaft), OpenSearch, FastAPI, Nginx, Selenium, Kiwipiepy.
- **특징**: Spark 기반 스트리밍 ETL과 Kafka 토픽 연동, OpenSearch 인덱싱 및 웹 검색 UI 연동, 컨테이너별 네트워크 분리와 마운트 기반 파일 입력 처리.

## B_project
- **설명**: 알고리즘 문제 풀이·코딩 테스트용 라이브러리와 자동화 도구 모음입니다.
- **기술**: Python(라이브러리/노트북), Jupyter Notebook, FastAPI/Async 백엔드(문서 참조), LLM 유틸리티 포함.
- **특징**: boj_bible(자료구조·그래프·문자열·고급알고리즘) 모듈, CT 자동화·LLM 연동 유틸리티(기본 모델 변경 안내 포함), kakao_history 등 예제 노트북 및 운영·개발 가이드 문서.
