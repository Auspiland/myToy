## geo_map
- **설명**: 사각형 영역 또는 좌표를 대한민국(남한/북한) 행정구역으로 변환하고 GeoJSON으로 생성·시각화하는 도구입니다.
- **기술**: Python, Shapely/STRtree, GeoPandas/GeoJSON, pandas
- **특징**: 고속 사각형→행정구역 변환(v2, 배치 처리 지원), 섬 지역 자동 감지 및 방향기반 대표 표현, 행정구역 경계 조회·GeoJSON 생성

## Mini-Pipeline
- **설명**: YouTube 스크립트 추출 → 문장 전처리 → Kafka와 OpenSearch로 업로드하는 E2E 스트리밍 파이프라인입니다.
- **기술**: Docker Compose, Apache Spark(Structured Streaming), Kafka(kRaft), OpenSearch, FastAPI/Nginx, Kiwipiepy
- **특징**: 컨테이너 기반 스트리밍 아키텍처(여러 서비스 네트워크 분리), Spark로 Kafka/OpenSearch 연동 및 실시간 전처리, Kiwi 기반 언어처리(udf) 및 디버깅용 콘솔 출력

## B_project
- **설명**: 알고리즘 문제 풀이·코딩 테스트 연습용 라이브러리 및 유틸 모음입니다.
- **기술**: Python, 자료구조·알고리즘 모듈, Jupyter Notebook
- **특징**: boj_bible(기본/그래프/트리/문자열/고급 알고리즘) 패키지, CT 도구(LLM 래퍼 포함) 및 예제 노트북(kakao_history), 운영·개발 가이드 문서(special_prompt.md)
