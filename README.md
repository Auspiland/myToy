## geo_map
- **설명**: 사각형 영역 또는 좌표를 대한민국(남한/북한) 행정구역(시/도 등)으로 변환하고 GeoJSON으로 시각화하는 도구입니다.  
- **기술**: Python, Shapely/STRtree 기반 공간 인덱스, GeoJSON 데이터, 공간 경계(국가/시도/군) 데이터.  
- **특징**: 사각형→행정구역 변환(대량 처리 지원), 좌표→행정구역 변환 및 섬 자동 감지, 행정구역 경계 조회·GeoJSON 생성 및 시각화.

## Mini-Pipeline
- **설명**: YouTube URL로부터 스크립트를 추출해 문장 전처리 후 Kafka·Opensearch로 업로드하는 컨테이너형 스트리밍 E2E 파이프라인입니다.  
- **기술**: Docker Compose, Spark Structured Streaming, Kafka(kRaft), OpenSearch, FastAPI/Nginx, Selenium, Kiwipiepy.  
- **특징**: 컨테이너 간 내부 네트워크 기반 스트리밍 파이프라인, Spark에서 Kafka/Opensearch로의 read/write 스트림 처리, 형태소 기반 전처리(Kiwi) 및 웹 검색 인터페이스 연동.

## B_project
- **설명**: 알고리즘 연습 및 코딩테스트용 라이브러리·도구 모음(문제 풀이 코드, 노트북, 자동화 도구).  
- **기술**: Python 기반 알고리즘 모듈(자료구조/그래프/트리/문자열/고급 알고리즘), Jupyter 노트북, FastAPI/LLM 유틸(일부 문서).  
- **특징**: boj_bible(자료구조·알고리즘 모듈) 제공, CT(코딩테스트) 자동화·기록 도구 및 LLM 호출 유틸 업데이트(기본 모델 변경), 예제 노트북 및 문제 풀이 함수 포함.
