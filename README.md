## geo_map
- **설명**: 좌표 또는 사각형 영역을 대한민국(남한/북한) 행정구역으로 변환하고 GeoJSON으로 시각화하는 도구입니다.
- **기술**: Python, shapely/GeoPandas 기반 공간 연산(STRtree 등), GeoJSON, 주요 데이터 파일(행정구역 GeoJSON).
- **특징**: 고속 사각형→행정구역 변환(rect2region_fast_v2), 섬 자동 감지 및 행정경계 조회, GeoJSON 생성 및 대량 JSON 일괄 변환/시각화.

## Mini-Pipeline
- **설명**: 유튜브 스크립트 추출 → 문장 전처리 → Kafka/Spark 스트리밍을 거쳐 OpenSearch에 업로드하는 컨테이너화된 E2E 파이프라인입니다.
- **기술**: Docker(멀티 컨테이너), Spark Structured Streaming, Kafka(kRaft), OpenSearch, FastAPI/Nginx, Kiwipiepy(한국어 전처리).
- **특징**: Spark+Kafka 기반 실시간 파이프라인(파일 모니터링 → 토픽 처리 → OpenSearch 적재), 컨테이너별 네트워크 구성으로 서비스 분리, Kiwipiepy와 룰베이스 전처리(오탈자 수정·문장 분리 등).

## B_project
- **설명**: 알고리즘/코딩 테스트 문제 풀이와 자동화 도구들을 모아둔 저장소입니다.
- **기술**: Python 기반 알고리즘 모듈(스택/그래프/트리/문자열/고급 알고리즘), Jupyter 노트북 및 유틸 스크립트, LLM 호출 유틸리티.
- **특징**: boj_bible로 정리된 알고리즘 라이브러리, CT 자동화 도구 및 예제(노트북 포함), LLM 연동 유틸(기본 모델이 gpt-5-nano로 변경된 점 포함).
