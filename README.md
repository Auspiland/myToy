## geo_map
- **설명**: 사각형 영역 또는 좌표를 대한민국 행정구역(광역·기초 단위)으로 변환하고 GeoJSON으로 시각화하는 파이썬 유틸리티입니다.
- **기술**: Python, Shapely/STRtree 공간 인덱스, GeoJSON, (geopandas/파이썬 공간처리 라이브러리)
- **특징**: 사각형→행정구역 변환(대량 처리 지원), 점→지역 매핑 및 경계 조회, 섬 감지·GeoJSON 생성 및 시각화

## Mini-Pipeline
- **설명**: Youtube 스크립트 수집부터 전처리, Kafka/Opensearch 업로드까지의 스트리밍 기반 E2E 파이프라인입니다.
- **기술**: Docker, Apache Spark (Structured Streaming), Kafka (kRaft), OpenSearch, FastAPI, Nginx, Selenium, Kiwipiepy
- **특징**: 컨테이너화된 스트리밍 파이프라인(Spark→Kafka→Opensearch), 텍스트 전처리(Kiwi·룰베이스) 및 웹 검색 인터페이스

## B_project
- **설명**: 알고리즘 문제 풀이 라이브러리와 코딩 테스트 자동화·기록 도구 모음입니다.
- **기술**: Python(알고리즘 구현, Jupyter 노트북), Async FastAPI/DB 도구 문서(운영 가이드), LLM 유틸리티 함수 포함
- **특징**: boj_bible(자료구조·알고리즘 모듈), CT 자동화(전처리·LLM 연동 유틸리티), 문제 풀이 예제·노트북 및 개발 운영 가이드
