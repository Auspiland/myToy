## geo_map
- **설명**: 사각형 영역 또는 좌표를 대한민국/북한 행정구역으로 변환하고 GeoJSON으로 시각화하는 파이썬 도구  
- **기술**: Python, Shapely/STRtree 기반 공간 인덱싱, GeoJSON, Geopandas 스타일 데이터 처리  
- **특징**: 사각형→행정구역 변환 및 대량 처리(convert_many), 좌표→행정구역 매핑(point2region), 섬 자동 감지·GeoJSON 생성 및 경계 조회

## Mini-Pipeline
- **설명**: YouTube 스크립트 추출부터 전처리·Kafka→Spark 스트리밍·Opensearch 색인까지의 E2E 스트리밍 파이프라인  
- **기술**: Docker Compose(컨테이너화), Spark Structured Streaming, Kafka(kRaft), Opensearch, FastAPI/Nginx, Kiwipiepy  
- **특징**: 파일 기반 스트리밍 입력 → Kafka 토픽 파이프라인, Kiwipiepy·룰 기반 문장 전처리(F.pandas_udf 적용), Opensearch 색인 및 웹 검색 인터페이스

## B_project
- **설명**: 알고리즘 연습 및 코딩 테스트 자동화·도구 모음(자료구조/알고리즘 라이브러리 포함)  
- **기술**: Python 기반 알고리즘 모듈, FastAPI/Async 백엔드 가이드 문서(참고용), Jupyter 노트북 예제  
- **특징**: boj_bible(기본 자료구조·그래프·트리·문자열·고급 알고리즘), CT 자동화 도구(LLM 호출 유틸 포함) 및 kakao_history.ipynb 예제, LLM 기본 모델이 gpt-5 → gpt-5-nano로 변경된 안내
