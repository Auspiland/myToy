## geo_map
- **설명**: 사각형 영역 또는 좌표를 대한민국(남한/북한) 행정구역으로 변환하고 GeoJSON으로 시각화하는 지리 공간 변환 툴입니다.  
- **기술**: Python, shapely/STRtree 기반 공간 인덱싱, GeoJSON, spatial 데이터 처리 도구들.  
- **특징**: 사각형→행정구역 변환(대량 처리 가능), 섬 자동 감지 및 방향 기반 대표 표현, 변환 결과의 GeoJSON 생성·조회·시각화.

## Mini-Pipeline
- **설명**: YouTube 스크립트 추출부터 전처리·Kafka·Spark 스트리밍·Opensearch 적재 및 웹 검색까지의 E2E 스트리밍 파이프라인입니다.  
- **기술**: Docker-compose(다중 네트워크), Spark Structured Streaming, Kafka(kRaft), OpenSearch, FastAPI, Nginx, Selenium, Kiwipiepy.  
- **특징**: 컨테이너화된 실시간 데이터 흐름(파일→Kafka→Spark→OpenSearch), 한국어 텍스트 전처리(Kiwi + 룰베이스, pandas_udf 적용), 웹 기반 검색 인터페이스 연동.

## B_project
- **설명**: 알고리즘 문제 풀이용 라이브러리 및 코딩 테스트 자동화·기록 도구 모음입니다.  
- **기술**: Python(알고리즘 모듈, Jupyter 노트북), FastAPI/Async 스택(문서화된 가이드), LLM 연동 유틸리티.  
- **특징**: BOJ용 알고리즘 모듈(자료구조·그래프·트리·문자열 등), CT 자동화 도구 및 LLM 호출 유틸(기본 모델 변경 안내), 문제 풀이 예제·노트북 포함.
