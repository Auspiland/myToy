## geo_map
- **설명**: 사각형 영역이나 좌표를 대한민국(남한/북한) 행정구역으로 변환하고 GeoJSON으로 시각화하는 도구입니다.  
- **기술**: Python, Shapely/STRtree, GeoPandas, GeoJSON, LLM(지명 번역), GitHub 연동.  
- **특징**: 고성능 사각형→행정구역 변환(rect2region_fast_v2), 섬 자동 감지 및 방향 기반 대표 표현, GeoJSON 생성/일괄 변환 및 경계 조회.

## Mini-Pipeline
- **설명**: YouTube 스크립트 추출부터 전처리·Kafka·Opensearch 업로드까지의 스트리밍 E2E 파이프라인입니다.  
- **기술**: Docker/Docker Compose, Apache Spark Structured Streaming, Kafka(kRaft), Opensearch, FastAPI, Nginx, Selenium, Kiwipiepy.  
- **특징**: 컨테이너별 네트워크 분리 및 streaming 처리, Spark readStream→Kafka→Opensearch 파이프라인, Kiwi 기반 문장 전처리(udf) 및 서비스 연동.

## B_project
- **설명**: 알고리즘/코딩 테스트 연습용 라이브러리와 자동화 도구(문제 풀이·기록·유틸)를 모아둔 프로젝트입니다.  
- **기술**: Python 기반 알고리즘 모듈(자료구조·그래프·트리·문자열 등), Jupyter 노트북, FastAPI/Async 관련 운영 가이드 문서.  
- **특징**: boj_bible 알고리즘 라이브러리, CT 자동화 도구(LLM 연동 유틸 포함) 및 kakao_history 노트북 예제, common_utils 기본 모델이 gpt-5 → gpt-5-nano로 변경 관련 안내.
