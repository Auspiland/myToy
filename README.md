## geo_map
- **설명**: 좌표 또는 사각형 영역을 한국 행정구역(시/도/시군구 등)으로 변환하고 GeoJSON으로 시각화하는 도구입니다.  
- **기술**: Python, Shapely/Geopandas, STRtree 공간색인, GeoJSON, GitHub 연동.  
- **특징**: 사각형↔행정구역 변환(대량 처리 지원), 좌표→행정구역 매핑 및 경계 조회, 섬 자동 감지·GeoJSON 생성 및 시각화.

## Mini-Pipeline
- **설명**: YouTube 스크립트 수집 후 전처리하여 Kafka와 Opensearch로 스트리밍 업로드하는 E2E 파이프라인입니다.  
- **기술**: Docker(Compose), Spark Structured Streaming, Kafka(kRaft), Opensearch, FastAPI/Nginx, Kiwipiepy, Selenium(수집 보조).  
- **특징**: 컨테이너 기반 스트리밍 처리(파일→Kafka→Opensearch), 텍스트 전처리(Kiwi·룰베이스) 및 Spark UDF 병렬화, 복수 네트워크/토폴로지 구성.

## B_project
- **설명**: 알고리즘 학습·코딩테스트 도구 모음(풀이 라이브러리 및 자동화 도구)입니다.  
- **기술**: Python 기반 알고리즘 모듈(자료구조·그래프·트리·문자열 등), Jupyter 노트북, FastAPI/LLM 관련 유틸(프로젝트 CT).  
- **특징**: boj_bible 알고리즘 라이브러리 제공, CT 자동화·LLM 연동 유틸(기본 모델 gpt-5-nano 변경 안내 포함), 문제 풀이 샘플 노트북 및 도구 설명.
