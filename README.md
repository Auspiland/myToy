## geo_map
- **설명**: 좌표 또는 사각형 영역을 대한민국(남한/북한) 행정구역으로 매핑하고 GeoJSON으로 출력/시각화합니다.  
- **기술**: Python, geopandas/shapely, STRtree 공간 인덱스, GeoJSON 처리.  
- **특징**: 사각형→행정구역(대량 배치 지원), 섬 자동 감지 및 대표 표현 규칙, 경계 조회·GeoJSON 생성 및 시각화 기능.

## Mini-Pipeline
- **설명**: YouTube 스크립트 추출부터 전처리→Kafka→Spark→Opensearch로 업로드하는 컨테이너화된 스트리밍 E2E 파이프라인입니다.  
- **기술**: Docker(멀티 컨테이너), Kafka(kRaft), Spark Structured Streaming, FastAPI, Opensearch, KiwiPy.  
- **특징**: 파일 모니터링→Kafka 토픽 처리→Opensearch 인덱싱 흐름, 한국어 전처리(오탈자·문장분리·불용어), Nginx/웹 UI 연동 검색 기능.

## B_project
- **설명**: 알고리즘 연습 및 코딩 테스트 도구 모음과 관련 문서·유틸을 제공합니다.  
- **기술**: Python(알고리즘 라이브러리, Jupyter 노트북), FastAPI 등 운영·배포 가이드 문서.  
- **특징**: boj_bible(자료구조·그래프·문자열·고급 알고리즘), CT 자동화·LLM 유틸(모델 기본값 변경 안내 포함), 문제 풀이 노트북 및 개발/배포 가이드 제공.
