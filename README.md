## geo_map
- **설명**: 사각형 영역 또는 좌표를 대한민국 행정구역(시/도/시군구 등)으로 변환하고 GeoJSON으로 시각화하는 도구입니다.
- **기술**: Python, shapely/GeoPandas, STRtree 공간 인덱스, GeoJSON 처리
- **특징**: 사각형→행정구역 변환(대표문구/상세), 섬 자동 감지 및 경계 조회, GeoJSON 파일 생성 및 일괄 JSON 변환

## Mini-Pipeline
- **설명**: 유튜브 스크립트 추출부터 문장 전처리, Kafka/Opensearch 저장까지의 컨테이너화된 스트리밍 파이프라인입니다.
- **기술**: Docker Compose, Apache Kafka(kRaft), Apache Spark Structured Streaming, FastAPI, Opensearch, Nginx
- **특징**: Streaming E2E 파이프라인(파일→Spark→Kafka→Opensearch), Kiwipiepy 기반 한국어 전처리(pandas_udf 활용), Docker 네트워크 분리 및 서비스별 컨테이너 구성

## B_project
- **설명**: 알고리즘 문제 풀이용 라이브러리와 코딩 테스트 자동화 도구 모음입니다.
- **기술**: Python(알고리즘 모듈, 노트북), FastAPI/Async 관련 유틸 문서(운영 가이드)
- **특징**: boj_bible(자료구조·알고리즘 모듈), CT 자동화 및 LLM 유틸(기본 모델이 gpt-5-nano로 변경된 함수 포함), kakao_history 노트북 및 운영·배포 관련 문서 포함
