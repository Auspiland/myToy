## geo_map
- **설명**: 한반도(남한/북한) 좌표 또는 사각형 영역을 행정구역으로 매칭하고 GeoJSON 생성·시각화하는 파이썬 도구.
- **기술**: Python, Shapely/STRtree 기반 공간 인덱싱, GeoJSON, Pip 패키지 기반 의존성.
- **특징**: Rect2Region 기반 사각형→행정구역 변환(단건/대량 처리), 섬 자동 감지 및 대표 텍스트 생성, 행정경계 조회·GeoJSON 파일 생성·시각화 및 배포.

## Mini-Pipeline
- **설명**: YouTube 스크립트 추출 → 전처리 → Kafka·Opensearch 업로드까지의 스트리밍 E2E 파이프라인(도커화).
- **기술**: Docker Compose, Apache Kafka(kRaft), Apache Spark Structured Streaming, FastAPI, Nginx, OpenSearch, Selenium, Kiwipiepy.
- **특징**: 컨테이너 기반의 스트리밍 아키텍처(Kafka↔Spark↔Opensearch), 한국어 전처리(키위·룰베이스)와 pandas_udf 적용, Nginx/웹 인터페이스로 검색·조회 제공.

## B_project
- **설명**: 알고리즘·코딩 테스트 학습용 모노레포(라이브러리, 자동화 도구, 노트북 등).
- **기술**: Python, Jupyter Notebook, 알고리즘 모듈(자료구조·그래프·트리·문자열·고급), CT용 LLM 유틸리티 및 FastAPI 관련 가이드.
- **특징**: boj_bible 알고리즘 라이브러리(기본·그래프·트리·문자열·고급), CT 자동화·기록 도구(LLM 응답 함수 포함) 및 문제 풀이 노트북, 운영·배포·개발 가이드 문서 제공.
