## geo_map
- **설명**: 좌표 또는 사각형 영역을 대한민국(남한/북한) 행정구역으로 변환해 대표 텍스트 및 상세 정보를 제공하는 도구  
- **기술**: Python, Shapely/STRtree 기반 공간 인덱싱, GeoJSON, GeoPandas 스타일 경계 데이터  
- **특징**: 사각형/좌표 → 행정구역 변환(대표문구/상세결과), 섬 자동 감지 및 방향·면적 기반 대표표현 규칙(95%/70%/40%), GeoJSON 생성·경계 조회·일괄 JSON 변환

## Mini-Pipeline
- **설명**: Youtube 스크립트 추출 → 문장 전처리 → Kafka/Opensearch 업로드로 구성된 E2E 스트리밍 파이프라인  
- **기술**: Docker(Compose), Spark Structured Streaming, Kafka(kRaft), Opensearch, FastAPI + Nginx, Selenium(스크립트 수집)  
- **특징**: 컨테이너 기반 멀티서비스 네트워크 구성, Spark readStream → Kafka/Opensearch 연동 처리, Kiwipiepy + pandas_udf 기반 텍스트 전처리(오탈자·문장분리·불용어)
