## geo_map
- **설명**: 사각형 영역이나 좌표를 대한민국(남한/북한) 행정구역으로 변환하고 GeoJSON으로 시각화하는 파이썬 유틸리티입니다.  
- **기술**: Python, Shapely/STRtree 기반 공간 인덱스, GeoJSON, Pandas 등 공간처리 라이브러리.  
- **특징**: 사각형→시/도/군/구 변환 (대량 처리 지원), 좌표→행정구역 변환 및 섬 자동 감지, 변환 결과를 GeoJSON으로 생성·시각화 및 일괄 처리.

## Mini-Pipeline
- **설명**: YouTube 스크립트 추출부터 문장 전처리하여 Kafka·Opensearch에 업로드하는 E2E 스트리밍 파이프라인입니다.  
- **기술**: Docker(Compose), Spark Structured Streaming, Kafka(kRaft), OpenSearch, FastAPI/Nginx, Kiwipiepy/Selenium.  
- **특징**: 컨테이너 기반 모듈화된 스트리밍 아키텍처, Spark로 Kafka 토픽 연동·전처리(F.pandas_udf 사용), Opensearch 인덱싱 및 웹 검색 인터페이스.

## B_project
- **설명**: 알고리즘 문제 풀이(백준용 유틸)와 코딩 테스트 자동화/기록 도구 모음 프로젝트입니다.  
- **기술**: Python, Jupyter Notebook, 알고리즘 라이브러리(자료구조·그래프·트리 등), FastAPI/DB 가이드 문서(운영 가이드 포함).  
- **특징**: boj_bible(자료구조·알고리즘 모듈), CT 자동화·기록 도구 및 LLM 유틸(pc: 기본 모델 변경 안내), 문제 풀이 예제(노트북 형태) 제공.
