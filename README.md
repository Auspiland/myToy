## geo_map
- **설명**: 사각형 영역 또는 좌표를 대한민국(남한/북한) 행정구역으로 변환하고 GeoJSON으로 출력/시각화하는 도구입니다.  
- **기술**: Python, Shapely/STRtree 기반 공간 인덱싱, GeoJSON 데이터, LLM(지명 번역 보조) 활용.  
- **특징**: 사각형→행정구역 변환(광역/기초단위), 행정구역 경계 조회 및 GeoJSON 생성·시각화, 섬 자동 감지 및 대량 JSON 일괄 변환 지원.

## Mini-Pipeline
- **설명**: YouTube 스크립트 추출 → 문장 전처리 → Kafka/Opensearch로 업로드하는 Docker 기반 E2E 스트리밍 파이프라인입니다.  
- **기술**: Docker Compose, Apache Spark(Structured Streaming), Kafka(kRaft), OpenSearch, FastAPI/Nginx, Kiwipiepy, Selenium(스크립트 수집 보조).  
- **특징**: Spark로 마운트 폴더 감시→Kafka 토픽 생산/소비→Opensearch 색인, Kiwipiepy 기반 텍스트 전처리(F.pandas_udf 사용), 컨테이너 네트워크 분리 및 운영 최적화.

## B_project
- **설명**: 알고리즘 풀이·코딩 테스트 연습용 라이브러리 및 자동화 도구 모음입니다.  
- **기술**: Python 중심(모듈화된 알고리즘 라이브러리), Jupyter 노트북, LLM 호출 유틸리티(자동 응답/스트리밍 지원).  
- **특징**: boj_bible(자료구조·그래프·트리·문자열·고급 알고리즘) 모듈, CT 자동화 도구(LLM 기본 모델 변경 등), kakao_history 등 예제 노트북 제공.
