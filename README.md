## geo_map
- **설명**: 좌표나 사각형 영역을 대한민국·북한 행정구역(시/도 등)으로 매핑하고 GeoJSON으로 시각화하는 도구입니다.
- **기술**: Python, Shapely/GEOS(STRtree 공간 인덱스), GeoJSON 기반 경계 데이터, (pandas 등 유틸성 라이브러리).
- **특징**: 사각형·좌표 → 행정구역 변환(배치 처리 지원), 섬 자동 감지 및 방향 기반 대표 표현, GeoJSON 생성·시각화 및 경계 조회 API.

## Mini-Pipeline
- **설명**: YouTube 스크립트 추출부터 전처리, Kafka·Spark를 거쳐 OpenSearch에 인덱싱되는 Docker화된 스트리밍 E2E 파이프라인입니다.
- **기술**: Docker Compose, Spark Structured Streaming, Kafka(kRaft), OpenSearch, FastAPI/Nginx, Kiwipiepy(한국어 전처리).
- **특징**: 컨테이너 기반 분산 스트리밍 아키텍처, Spark ↔ Kafka ↔ OpenSearch 연계 처리, Kiwi 기반 문장 전처리(오탈자·분절·불용어 처리).

## B_project
- **설명**: 알고리즘 문제 풀이용 라이브러리와 코딩 테스트 자동화·연습 도구 모음입니다.
- **기술**: Python 중심(모듈화된 알고리즘 패키지, Jupyter 노트북 포함).
- **특징**: boj_bible(자료구조·그래프·트리·문자열·고급 알고리즘) 모듈, CT 자동화 도구(LLM 연동 기본 모델 변경 등) 및 kakao_history 노트북의 문제 풀이 유틸 추가.
