## geo_map
- **설명**: 사각형 영역이나 좌표를 대한민국(남한/북한) 행정구역(시/도/시군구 등)으로 변환하고 GeoJSON으로 시각화하는 도구입니다.
- **기술**: Python, Shapely/GeoPandas(공간 인덱스 STRtree), GeoJSON, 공간 데이터 처리 스크립트.
- **특징**: 사각형→행정구역 변환(대량 처리 지원), 좌표→지역 매핑 및 경계 조회, 섬 자동 감지·GeoJSON 생성 및 시각화.

## Mini-Pipeline
- **설명**: YouTube URL로부터 스크립트를 추출해 전처리하고 Kafka/Spark를 거쳐 OpenSearch에 적재하는 컨테이너화된 스트리밍 E2E 파이프라인입니다.
- **기술**: Docker Compose, Spark Structured Streaming, Kafka(kRaft), OpenSearch, FastAPI, Nginx, Selenium, Kiwipiepy.
- **특징**: 실시간(Streaming) 데이터 플로우(Kafka↔Spark↔OpenSearch), 컨테이너 네트워크 기반 서비스 연동, 한국어 전처리(Kiwipiepy)·검색/웹 노출 연동.

## B_project
- **설명**: 알고리즘 문제 풀이·코딩테스트용 라이브러리 및 자동화 도구 모음입니다.
- **기술**: Python 기반 알고리즘 구현(자료구조·그래프·트리·문자열 등), Jupyter 노트북 및 유틸 스크립트, LLM 연동 유틸.
- **특징**: boj_bible(자료구조·알고리즘 모듈), CT(코딩테스트 자동화·기록 도구) 및 예제 노트북, LLM 헬퍼 함수·운영 가이드 문서.
