## geo_map
- **설명**: 사각형 영역 또는 단일 좌표를 대한민국(남한/북한) 행정구역으로 매핑하고 GeoJSON 생성·시각화하는 툴입니다.
- **기술**: Python, Shapely/GeoPandas(STRtree), GeoJSON 처리, 공간 인덱스 기반 최적화
- **특징**: 사각형→행정구역 변환(rect2region_fast_v2), 좌표 기반 지역 조회 및 경계 반환, GeoJSON 생성·섬 자동 감지

## Mini-Pipeline
- **설명**: YouTube 스크립트 수집 → 문장 전처리 → Kafka/Spark 스트리밍 → OpenSearch 업로드까지의 컨테이너화된 E2E 파이프라인입니다.
- **기술**: Docker, Apache Spark(PySpark), Kafka(kRaft), OpenSearch, FastAPI, Nginx, Selenium, Kiwipiepy
- **특징**: 컨테이너 기반 스트리밍 아키텍처, Spark 스트림으로 Kafka/Opensearch 연동 및 전처리 파이프라인, 웹 검색 인터페이스(Nginx + FastAPI)

## B_project
- **설명**: 백준 알고리즘 풀이와 코딩 테스트 자동화/기록을 위한 파이썬 기반 프로젝트 모음입니다.
- **기술**: Python, Jupyter Notebook, 알고리즘/자료구조 라이브러리, CT 자동화 유틸(LLM 연동 포함)
- **특징**: BOJ용 자료구조·알고리즘 모듈(basic/graph/tree/string/advanced), 코딩 테스트 자동화 도구 및 예제 노트북, LLM 유틸(common_utils) 기본 모델 변경 안내(gpt-5 → gpt-5-nano)
