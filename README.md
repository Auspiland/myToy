## geo_map
- **설명**: 사각형 영역 또는 단일 좌표를 대한민국/북한 행정구역(시/도/시군구 등)으로 변환하고 시각화하는 툴입니다.  
- **기술**: Python, GeoJSON, Shapely/STRtree 기반 공간 인덱스, 공간 데이터(GeoJSON) 처리.  
- **특징**: 사각형→행정구역 변환(대량 처리와 성능 최적화), 섬 자동 감지 및 방향 기반 대표 표현, GeoJSON 생성·경계 조회 기능.

## Mini-Pipeline
- **설명**: 유튜브 스크립트 추출부터 문장 전처리, Kafka/Opensearch 업로드까지의 E2E 스트리밍 파이프라인입니다.  
- **기술**: Docker Compose, Spark Structured Streaming, Kafka(kRaft), OpenSearch, FastAPI/Nginx, Kiwipiepy(문장 전처리).  
- **특징**: 컨테이너 기반 멀티서비스 네트워크 구성(Streaming mode), Spark↔Kafka↔OpenSearch 흐름으로 실시간 파이프라인 처리, Kiwi 기반 텍스트 전처리(Pandas UDF) 및 모니터링용 콘솔 출력.

## B_project
- **설명**: 알고리즘 문제 풀이·코딩 테스트 보조 라이브러리와 관련 도구들을 모아둔 프로젝트입니다.  
- **기술**: Python(알고리즘 모듈, Jupyter 노트북), 유틸 스크립트 및 FastAPI/서비스 설계 문서(참고문서).  
- **특징**: boj_bible(자료구조·알고리즘 모듈) 구성, CT(코딩 테스트) 자동화·기록 도구 및 예제 노트북 포함, LLM 연동 유틸(기본 모델 변경 등) 문서화.
