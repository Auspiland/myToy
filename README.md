## geo_map
- **설명**: 사각형 영역 또는 단일 좌표를 대한민국(남한/북한) 행정구역으로 변환하고 GeoJSON으로 시각화하는 도구입니다.
- **기술**: Python, Shapely/STRtree 기반 공간 인덱스, GeoJSON, GitHub 연동 스크립트
- **특징**: 빠른 사각형→행정구역 변환(rect2region_fast_v2), 행정구역 경계 조회 및 GeoJSON 생성/시각화, 섬 자동 감지 및 방향 기반 대표 표현

## Mini-Pipeline
- **설명**: YouTube 스크립트 추출부터 전처리·스트리밍·검색 인덱싱까지의 E2E 데이터 파이프라인입니다.
- **기술**: Docker(Compose), Apache Spark Streaming, Kafka(kRaft), OpenSearch, FastAPI/Nginx, Kiwipiepy
- **특징**: 컨테이너 기반 스트리밍 파이프라인(Kafka ↔ Spark ↔ OpenSearch), 한국어 전처리(Kiwipiepy + pandas_udf) 적용, 웹 검색 인터페이스(Nginx→FastAPI)

## B_project
- **설명**: 알고리즘 풀이 및 코딩 테스트 자동화·학습을 위한 유틸 모음입니다.
- **기술**: Python(알고리즘 라이브러리), 코딩 테스트 자동화 도구, LLM 호출 유틸리티
- **특징**: boj_bible(자료구조·그래프·문자열·고급 알고리즘) 모듈, CT 자동화 및 기록 도구, LLM 관련 유틸(기본 모델이 gpt-5-nano로 변경) 및 문제풀이 예제 노트북 추가
