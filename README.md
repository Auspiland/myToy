## geo_map
- **설명**: 좌표나 사각형 영역을 대한민국/북한 행정구역으로 변환하고 경계·GeoJSON을 생성해 시각화하는 도구입니다.
- **기술**: Python, GeoJSON, Shapely/STRtree 기반 공간 인덱싱, 지오데이터(GeoJSON/Shapefile) 처리 도구들.
- **특징**: 고성능 사각형→행정구역 변환(rect2region_fast_v2), 섬 자동 감지 및 대표 표현 규칙, 행정구역 경계 조회 및 GeoJSON 생성/배포.

## Mini-Pipeline
- **설명**: YouTube 스크립트 추출부터 문장 전처리·Kafka·OpenSearch 업로드까지 Docker 기반 스트리밍 E2E 파이프라인입니다.
- **기술**: Docker/compose, Spark Structured Streaming, Kafka(kRaft), OpenSearch, FastAPI, Nginx, Kiwipiepy, Selenium(수집).
- **특징**: 마운트 폴더 감시→Spark로 Kafka 토픽 생성→가공 후 OpenSearch에 색인, Kiwi 기반 전처리(Pandas UDF) 및 컨테이너별 네트워크 분리 구성.

## B_project
- **설명**: 알고리즘 학습·코딩 테스트 솔루션과 관련 도구들을 모아둔 프로젝트입니다.
- **기술**: Python 기반 알고리즘 모듈 및 Jupyter 노트북(문제 풀이 예제), FastAPI/Async 관련 운영 가이드 문서 포함.
- **특징**: boj_bible(기초/그래프/트리/문자열/고급 알고리즘) 모듈, CT 자동화·LLM 연동 유틸(기본 모델 변경 안내), kakao_history 노트북의 문제 풀이 예제 포함.
