## geo_map
- **설명**: 사각형 영역 또는 좌표를 대한민국(남한/북한) 행정구역으로 변환하고 해당 경계/GeoJSON을 생성하는 툴입니다.  
- **기술**: Python, GeoJSON, Shapely/pyproj, STRtree 공간 인덱스 기반 탐색.  
- **특징**: 사각형→행정구역 변환(대량 처리 지원), 섬 자동 감지 및 방향 기반 대표 표현, 경계 조회·GeoJSON 생성 및 일괄 변환 기능.

## Mini-Pipeline
- **설명**: YouTube URL에서 스크립트를 추출해 문장 전처리 후 Kafka/Opensearch로 업로드하는 E2E 스트리밍 파이프라인입니다.  
- **기술**: Docker(Compose), Spark Structured Streaming, Kafka(kRaft), Opensearch, FastAPI/Nginx, Kiwipiepy, Selenium(스크랩).  
- **특징**: 컨테이너 기반 네트워크 분리(서비스별 compose), Spark로 파일/스트림 처리해 Kafka·Opensearch에 적재, 전처리(오탈자 수정·문장 분리 등)를 위한 UDF/Kiwi 통합.

## B_project
- **설명**: 알고리즘 문제 풀이 및 코딩 테스트 준비를 위한 자료구조·알고리즘 라이브러리와 도구 모음입니다.  
- **기술**: Python 중심(모듈화된 알고리즘 구현, Jupyter 노트북 포함), FastAPI/기타 운영 가이드 문서(참고용).  
- **특징**: boj_bible(기본 자료구조·그래프·트리·문자열·고급 알고리즘) 모듈, CT 관련 자동화·LLM 유틸(기본 모델 변경 등) 및 kakao_history 등 예제 노트북 제공.
