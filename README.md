## geo_map
- **설명**: 한반도 좌표 또는 사각형 영역을 행정구역(시/도/시군구) 단위로 변환하고 GeoJSON으로 시각화하는 도구  
- **기술**: Python, geopandas/shapely, STRtree 공간 인덱스, GeoJSON 출력  
- **특징**: 사각형→행정구역 대량 변환(대표 텍스트/상세), 섬 지역 자동 감지·경계 조회, GeoJSON 생성 및 결과 시각화/배포

## Mini-Pipeline
- **설명**: YouTube 스크립트 추출부터 전처리하여 Kafka/Spark를 거쳐 Opensearch에 업로드하는 컨테이너화된 스트리밍 파이프라인  
- **기술**: Docker Compose, Spark Structured Streaming, Kafka(kRaft), Opensearch, FastAPI/Nginx, Kiwipiepy, Selenium  
- **특징**: 폴더 감시 → Spark→Kafka→Opensearch E2E 스트리밍, 한국어 전처리(Kiwi·규칙기반) 및 pandas_udf 적용, 컨테이너 네트워크·배포 구성

## B_project
- **설명**: 알고리즘 문제 풀이·코딩 테스트용 라이브러리 및 보조 도구 모음 저장소  
- **기술**: Python, 알고리즘 모듈(그래프/트리/문자열/고급), Jupyter 노트북, FastAPI/LLM 유틸 문서  
- **특징**: boj_bible 알고리즘 라이브러리 구조화, CT 자동화·LLM 연동 유틸(모델 기본값 변경 기록 포함), 예제 노트북 및 운영·개발 가이드 문서
