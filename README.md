## geo_map
- **설명**: 사각형 영역 또는 좌표를 대한민국(남·북한 포함) 행정구역으로 매핑하고 GeoJSON으로 시각화하는 도구입니다.
- **기술**: Python, Shapely/STRtree 기반 공간 색인, GeoJSON, Pandas 등 GIS 관련 라이브러리.
- **특징**: 사각형→행정구역 변환(고성능 rect2region_fast_v2), 좌표→행정구역 변환 및 경계 조회, 섬 자동 감지·GeoJSON 파일 생성 및 일괄 변환.

## Mini-Pipeline
- **설명**: YouTube 스크립트를 수집·전처리해 Kafka와 Opensearch로 스트리밍 적재하는 컨테이너화된 E2E 파이프라인입니다.
- **기술**: Docker Compose, Spark Structured Streaming, Kafka(kRaft), Opensearch, FastAPI/Nginx, Kiwipiepy(전처리).
- **특징**: Spark→Kafka→Opensearch로 이어지는 실시간 스트리밍 처리, Kiwipiepy 기반 문장 전처리(pandas_udf 사용), 컨테이너 간 네트워크·오케스트레이션 구성 및 로깅/디버깅용 writeStream 콘솔 활용.

## B_project
- **설명**: 알고리즘 연습(백준 풀이)과 코딩 테스트 자동화 도구들을 모아둔 프로젝트입니다.
- **기술**: Python(알고리즘 라이브러리), Jupyter Notebook, FastAPI/Async 구성 문서(운영 가이드), LLM 유틸리티.
- **특징**: boj_bible(기본 자료구조·그래프·트리·문자열 등 알고리즘 모듈), CT 자동화·기록 도구(LLM 호출 유틸 포함, 기본 모델이 gpt-5-nano로 변경됨 알림), 문제별 예제 노트북 및 개발·배포 가이드 문서 보유.
