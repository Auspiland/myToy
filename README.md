## geo_map
- **설명**: 사각형 영역 또는 좌표를 대한민국(남한/북한) 행정구역으로 변환하고 GeoJSON으로 시각화하는 파이썬 유틸리티.
- **기술**: Python, geopandas/shapely, STRtree 공간 인덱스, GeoJSON, LLM(지명 번역 보조).
- **특징**: rect2region_fast_v2 기반 고성능 사각형→행정구역 변환; 좌표→지역 매핑 및 행정구역 경계 조회; GeoJSON 생성·시각화·일괄 변환(섬 자동 감지 포함).

## Mini-Pipeline
- **설명**: YouTube 스크립트 수집 → 문장 전처리 → Kafka/Opensearch 업로드까지의 Docker 기반 스트리밍 E2E 파이프라인.
- **기술**: Docker Compose, Spark Structured Streaming, Kafka(kRaft), OpenSearch, FastAPI/Nginx, Kiwipiepy(문장분석).
- **특징**: 마운트 폴더 감시 기반 스트리밍 입력 → Kafka 토픽 및 OpenSearch로 출력; Spark에서 pandas_udf/Transfer로 한국어 전처리(오탈자·문장분리 등); 컨테이너화된 웹 검색 인터페이스 제공.

## B_project
- **설명**: 알고리즘/코딩테스트 연습용 라이브러리 및 자동화 도구 모음.
- **기술**: Python 기반 알고리즘 구현(그래프/트리/문자열/고급 알고리즘), Jupyter 노트북, FastAPI/Async 관련 안내 문서.
- **특징**: boj_bible(자료구조·알고리즘 모듈) 제공; CT 자동화·기록 도구(LLM 연동 함수 포함) 및 문제 풀이 노트북 추가; 운영·배포 가이드(special_prompt.md) 포함.
