## geo_map
- **설명**: 사각형 영역 또는 좌표를 대한민국 행정구역(광역/시군구)으로 변환하고 GeoJSON으로 시각화하는 도구입니다.  
- **기술**: Python, Shapely/STRtree 기반 공간 인덱스, GeoJSON, 경계 데이터(GeoJSON) 처리.  
- **특징**: 사각형→행정구역 변환(대량 처리 최적화), 섬 자동 감지 및 대표 표현 생성, 행정구역 경계 조회·GeoJSON 생성/배포.

## Mini-Pipeline
- **설명**: YouTube 스크립트 추출부터 전처리·Kafka·Spark·Opensearch로의 스트리밍 E2E 파이프라인입니다.  
- **기술**: Docker-compose(컨테이너화), Spark Structured Streaming, Kafka(kRaft), Opensearch, FastAPI/Nginx, Kiwipiepy(한국어 전처리).  
- **특징**: 컨테이너 기반 실시간 스트리밍 파이프라인, Spark→Kafka→Opensearch 흐름으로 색인·검색 제공, Kiwipiepy 기반 오탈자 수정·문장 분리 등 전처리 파이프라인.

## B_project
- **설명**: 알고리즘 연습 및 코딩테스트 솔루션 라이브러리와 도구 모음입니다.  
- **기술**: Python(알고리즘 구현, Jupyter 노트북), 테스트용 유틸 및 LLM 호출 유틸리티(CT 도구군).  
- **특징**: boj_bible(자료구조·그래프·트리·문자열 등 알고리즘 모듈), CT 자동화 도구(LLM 래퍼 포함) 및 문제풀이 노트북·운영 가이드 문서.
