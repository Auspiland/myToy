## geo_map
- **설명**: 좌표(단일/사각형)나 영역을 대한민국(남·북한 포함) 행정구역 명칭으로 변환하고 GeoJSON으로 시각화하는 도구입니다.
- **기술**: Python, GeoPandas/Shapely, STRtree 공간 인덱스, GeoJSON 출력을 지원하는 파이프라인.
- **특징**: 사각형→행정구역 변환(대표표현/상세), 좌표→행정구역 변환 및 경계 조회, 섬 자동 감지·GeoJSON 생성 및 일괄 변환 지원.

## Mini-Pipeline
- **설명**: 유튜브 스크립트를 입력으로 문장 전처리하여 Kafka와 OpenSearch로 스트리밍 적재하는 엔드투엔드 파이프라인입니다.
- **기술**: Docker(Compose), Apache Spark Structured Streaming, Kafka(kRaft), OpenSearch, FastAPI, Nginx, Kiwipiepy.
- **특징**: 컨테이너 기반 스트리밍 파이프라인(파일 → Spark → Kafka → OpenSearch), Kiwi 기반 텍스트 전처리(udf/pandas_udf), 멀티컨테이너 네트워킹 및 모니터링 구조.

## B_project
- **설명**: 알고리즘/코딩테스트 풀이용 라이브러리와 자동화 도구(노트북·유틸 포함) 모음입니다.
- **기술**: Python 기반 알고리즘 모듈, Jupyter 노트북, FastAPI/비동기 유틸 문서(참고용).
- **특징**: boj_bible(자료구조·그래프·트리·문자열 등) 모듈, CT 자동화 및 LLM 호출 유틸(기본 모델 변경 안내 포함), 문제 풀이 노트북·운영 가이드 문서 제공.
