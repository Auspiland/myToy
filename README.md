## [geo_map]
- **설명**: 좌표 또는 사각형 영역을 대한민국(남북한) 행정구역으로 변환하고 GeoJSON으로 시각화하는 도구.  
- **기술**: Python, GeoJSON, Shapely/STRtree 기반 공간 인덱싱, 공간 경계 데이터(GeoJSON).  
- **특징**: 사각형→행정구역(고성능 v2), 섬 자동 감지 및 대표 표현 규칙, 경계 조회·GeoJSON 생성·대량 JSON 변환 지원.

## [Mini-Pipeline]
- **설명**: 유튜브 스크립트 추출부터 전처리→Kafka→Opensearch 업로드까지의 스트리밍 E2E 파이프라인.  
- **기술**: Docker Compose(다중 컨테이너 네트워크), Spark Structured Streaming, Kafka(kRaft), Opensearch, FastAPI/Nginx, Selenium/Kiwipiepy.  
- **특징**: 컨테이너 기반 네트워크 아키텍처(서비스별 compose), Spark로 파일 감시→토픽/인덱스 처리, 한국어 전처리(Kiwipiepy)·pandas_udf 최적화.

## [B_project]
- **설명**: 알고리즘 학습 및 코딩 테스트용 도구와 예제 모음(라이브러리·노트북 포함).

- **특징**: boj_bible(자료구조·그래프·트리·문자열·고급 알고리즘) 모듈, CT 자동화·LLM 호출 유틸(기본 모델 변경 안내 포함), kakao_history.ipynb 등 실전 풀이 예제.
