## geo_map
- **설명**: 좌표 또는 사각형 영역을 대한민국(남한/북한) 행정구역으로 빠르게 변환해주는 도구입니다.
- **기술**: Python, Shapely(STRtree), GeoJSON, geopandas 기반 처리 및 간단한 LLM 활용(지명 번역).
- **특징**: 사각형→행정구역 고속 변환(대표 표현/상세 결과), 좌표→행정구역 및 경계 조회, GeoJSON 생성·시각화 및 섬 자동 감지.

## Mini-Pipeline
- **설명**: YouTube 스크립트를 수집하여 전처리하고 Kafka/Spark를 통해 OpenSearch에 업로드하는 스트리밍 E2E 파이프라인입니다.
- **기술**: Docker(Compose), Kafka(kRaft), Spark(Structured Streaming), FastAPI, Nginx, OpenSearch, Kiwipiepy.
- **특징**: 컨테이너화된 스트리밍 파이프라인(ingest→transform→index), Spark 기반 실시간 전처리(Kiwi·룰베이스), 웹 검색(오픈서치 연동) 인터페이스 제공.

## B_project
- **설명**: 알고리즘 학습 및 코딩 테스트 자동화에 도움이 되는 라이브러리·도구 모음입니다.
- **기술**: Python 기반 알고리즘 모듈, Jupyter 노트북 예제, FastAPI/LLM 유틸(문서화 참조).
- **특징**: boj_bible(자료구조·그래프·문자열·고급 알고리즘) 모듈, CT 자동화/LLM 유틸(모델 기본 파라미터 변경 안내 포함), 문제 풀이 노트북 예제 제공.
