## geo_map
- **설명**: 사각형 영역 또는 좌표를 대한민국(남한/북한) 행정구역 단위로 매핑하고 GeoJSON으로 출력하는 도구  
- **기술**: Python, Shapely/STRtree 기반 공간 인덱스, GeoJSON, GitHub 자동 푸시 스크립트  
- **특징**: 사각형→행정구역 변환(대량 처리 지원), 섬 자동 감지 및 방향 기반 대표 표현, 경계 조회·GeoJSON 생성 및 일괄 변환

## Mini-Pipeline
- **설명**: YouTube 스크립트 추출부터 전처리·Streaming으로 Kafka/Spark를 거쳐 OpenSearch에 업로드하는 E2E 파이프라인  
- **기술**: Docker Compose(컨테이너화), Kafka(kRaft), Spark Structured Streaming, FastAPI/Nginx, OpenSearch, Selenium·Kiwipiepy 전처리  
- **특징**: 컨테이너 기반 스트리밍 아키텍처(다중 네트워크), Spark로 파일→Kafka→OpenSearch 파이프라인 구현, 한국어 전처리(오탈자·문장분리) 최적화

## B_project
- **설명**: 알고리즘 학습·코딩테스트 보조용 코드베이스 및 실습 도구 모음  
- **기술**: Python(라이브러리/노트북), 문제별 알고리즘 구현(그래프·트리·문자열·고급 알고리즘), Jupyter 노트북  
- **특징**: boj_bible 모듈화된 알고리즘 모음, CT 자동화·문서화 도구 및 예제 노트북 제공, LLM 연동 유틸(기본 모델 변경 및 스트리밍/폴백 지원)
