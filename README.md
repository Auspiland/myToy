## geo_map
- **설명**: 사각형 영역이나 좌표를 한국(남한/북한) 행정구역으로 빠르게 변환하고 GeoJSON으로 시각화하는 툴
- **기술**: Python, Shapely/STRtree 기반 공간 색인, GeoJSON, GitHub 연동
- **특징**: 사각형→행정구역 변환(고속 버전), 좌표→행정구역 매핑 및 경계 조회, GeoJSON 생성·시각화 및 섬 지역 자동 감지

## Mini-Pipeline
- **설명**: YouTube 스크립트 추출 → 문장 전처리 → Kafka/Opensearch로 업로드하는 E2E 스트리밍 파이프라인
- **기술**: Docker(멀티 컨테이너, docker-compose), Spark Structured Streaming, Kafka(kRaft), OpenSearch, FastAPI/Nginx, Kiwipiepy
- **특징**: Spark로 파일 감시·스트리밍 처리하여 Kafka/Opensearch로 전달, Kiwipiepy 기반 텍스트 전처리(오탈자·문장분리 등), 웹 검색 인터페이스(Nginx+FastAPI) 연동

## B_project
- **설명**: 알고리즘 연습·코딩테스트 지원 라이브러리 및 관련 도구 모음
- **기술**: Python(알고리즘 구현, Jupyter 노트북), 표준 알고리즘·자료구조 모듈, LLM 유틸리티(응답 함수)
- **특징**: boj_bible(자료구조·그래프·문자열 등 알고리즘 모듈), CT(코딩테스트 자동화·노트북 예제), LLM 관련 유틸(기본 모델 파라미터 변경 등)
