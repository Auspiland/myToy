## geo_map
- **설명**: 사각형 영역 또는 좌표를 대한민국(남한/북한) 행정구역 명칭으로 변환하고 경계/GeoJSON을 생성·시각화하는 도구입니다.  
- **기술**: Python, GeoJSON, shapely / STRtree 기반 공간 인덱싱, 지리공간 데이터(Shapefile/GeoJSON).  
- **특징**: 사각형→행정구역·좌표→행정구역 변환(고속), 섬 지역 자동 감지 및 대표 표현 생성, 행정구역 경계 조회·GeoJSON 출력 및 일괄 변환.

## Mini-Pipeline
- **설명**: YouTube 스크립트 추출부터 문장 전처리, Kafka·OpenSearch 업로드까지의 E2E 스트리밍 파이프라인입니다.  
- **기술**: Docker(Compose), Spark Structured Streaming, Kafka(kRaft), OpenSearch, FastAPI + Nginx, Kiwipiepy(한국어 전처리), Selenium(스크립트 수집 보조).  
- **특징**: Spark → Kafka → OpenSearch 흐름의 스트리밍 처리(파일 감시 기반 입력), Kiwipiepy+pandas_udf 기반 문장 전처리(오탈자/토큰화/불용어), Docker로 컨테이너화된 서비스와 웹 검색 인터페이스 연동.

## B_project

- **기술**: Python 기반 알고리즘 모듈(자료구조·그래프·트리·문자열 등), Jupyter 노트북, CT용 유틸리티(LLM 호출 헬퍼 포함).  
- **특징**: boj_bible(자료구조·알고리즘 라이브러리), CT 자동화·문제 풀이 유틸 및 예제 노트북(kakao_history), LLM 연동 함수 및 운영·배포 가이드 문서(special_prompt.md).
