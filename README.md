## geo_map
- **설명**: 좌표나 사각형 영역을 대한민국(남/북 포함) 행정구역으로 변환하고 GeoJSON으로 시각화하는 툴입니다.  
- **기술**: Python, Shapely/GeoPandas, STRtree 공간 인덱스, GeoJSON 출력.  
- **특징**: 사각형→행정구역 변환(빠른 v2), 좌표→행정구역 변환 및 경계 조회, 섬 자동 감지·GeoJSON 생성 및 일괄 변환.

## Mini-Pipeline
- **설명**: YouTube 스크립트 추출부터 문장 전처리, Kafka·Opensearch 업로드까지의 도커 기반 스트리밍 파이프라인입니다.  
- **기술**: Docker Compose, Spark Structured Streaming, Kafka(kRaft), OpenSearch, FastAPI/Nginx, Kiwipiepy(한국어 전처리).  
- **특징**: E2E 스트리밍 파이프라인(파일→Spark→Kafka→OpenSearch), 컨테이너 네트워크 구성 및 장애 회피, 웹 검색 인터페이스 연동.

## B_project
- **설명**: 알고리즘 문제 풀이 및 코딩 테스트 지원용 라이브러리·도구 모음입니다.  
- **기술**: Python(모듈·노트북), 알고리즘 구현 라이브러리, LLM 연동 유틸(CT 자동화).  
- **특징**: boj_bible(자료구조·알고리즘 모듈), CT 자동화 도구 및 LLM 기반 헬퍼 변경사항(기본 모델 gpt-5-nano), 교육용 노트북·운영 가이드 문서 포함.
