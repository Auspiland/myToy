## geo_map
- **설명**: 사각형 영역 또는 좌표를 대한민국 행정구역(시/도/시군구 등)으로 변환하고 GeoJSON으로 시각화하는 파이썬 유틸리티.
- **기술**: Python, Shapely(STRtree) 기반 공간 인덱싱, GeoJSON/GeoPandas, LLM(지명 번역) 보조.
- **특징**: Rect2Region(v2) 기반 고속 변환(대량 처리), 섬 자동 감지·방향 기반 대표 표현, 행정구역 경계 조회 및 GeoJSON 생성/일괄 변환 지원.

## Mini-Pipeline
- **설명**: YouTube 스크립트 추출부터 문장 전처리·Kafka 전송·OpenSearch 적재까지의 Docker 기반 스트리밍 E2E 파이프라인.
- **기술**: Docker, Apache Spark Structured Streaming, Kafka(kRaft), OpenSearch, FastAPI, Nginx, Kiwipiepy, Selenium.
- **특징**: 컨테이너화된 마이크로서비스 아키텍처(Streaming mode), Spark↔Kafka↔OpenSearch 연동 및 Kiwi 기반 한국어 전처리(UDF), 웹 인터페이스를 통한 검색/시각화 제공.

## B_project
- **설명**: 알고리즘 학습·코딩테스트 솔루션 모음과 자동화 도구, 운영 가이드 등을 포함한 레포지토리.
- **기술**: Python(알고리즘 모듈·노트북), FastAPI/Async, SQLAlchemy(Async)/Redis, Docker 및 CI/CD 가이드 문서화.
- **특징**: boj_bible(자료구조·알고리즘 모듈) 중심 코드, CT 자동화·LLM 헬퍼(기본 모델 변경 안내 포함), 문제 풀이 노트북 및 배포/운영 문서 제공.
