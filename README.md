## geo_map
- **설명**: 한반도(남·북) 좌표 또는 사각형 영역을 행정구역(시/도/시군구 등)으로 변환하고 GeoJSON으로 생성·시각화하는 파이썬 도구입니다.
- **기술**: Python, Shapely/STRtree 기반 공간인덱스, GeoJSON, GitHub 연동 스크립트.
- **특징**: 사각형→행정구역 대량 변환(convert_many)·대표 문구 생성, 섬 자동 감지(지역별 임계값), 행정구역 경계 조회 및 GeoJSON 출력/시각화.

## Mini-Pipeline
- **설명**: YouTube 스크립트 추출·전처리 후 Kafka → Spark → Opensearch로 스트리밍 업로드하는 컨테이너형 E2E 파이프라인입니다.
- **기술**: Docker Compose, Apache Kafka(kRaft), Apache Spark(Structured Streaming), Opensearch, FastAPI, Nginx, Selenium, Kiwipiepy.
- **특징**: 마운트 폴더 감시 및 Streaming 파이프라인(파일→Kafka→Opensearch), Kiwi 기반 문장 전처리(F.pandas_udf)·룰베이스 오탈자 수정, 컨테이너 네트워크 설계 및 장애(타이밍) 대비 처리.

## B_project
- **설명**: 알고리즘/코딩 테스트 풀이 모음과 코딩 테스트 자동화·도구 문서를 포함한 학습용 프로젝트입니다.
- **기술**: Python(알고리즘 라이브러리, Jupyter 노트북), FastAPI/Async·SQLAlchemy/Redis(문서화된 가이드), LLM 연동 유틸리티.
- **특징**: boj_bible(자료구조·그래프·문자열·고급 알고리즘) 모듈, CT 자동화 도구 및 kakao_history 노트북 예제(문제별 솔루션), LLM 호출 유틸의 기본 모델 변경 등 운영 이력 관리.
