## geo_map
- **설명**: 사각형 영역 또는 좌표를 대한민국(남한/북한) 행정구역으로 변환하고 GeoJSON 생성 및 시각화하는 툴.
- **기술**: Python, Shapely/STRtree, GeoPandas/GeoJSON 등 공간 데이터 처리 스택.
- **특징**: 사각형→행정구역 변환(고성능 rect2region_fast_v2), 좌표→행정구역 변환 및 경계 조회, 섬 자동 감지·GeoJSON 일괄 생성/출력.

## Mini-Pipeline
- **설명**: YouTube 스크립트 추출·전처리 후 Kafka·Spark·Opensearch에 실시간 업로드하는 컨테이너 기반 E2E 스트리밍 파이프라인.
- **기술**: Docker, Spark Structured Streaming, Kafka(kRaft), OpenSearch, FastAPI/Nginx, Kiwipiepy(한국어 전처리), Selenium(수집 보조).
- **특징**: Streaming 모드의 엔드투엔드 처리(Kafka 토픽 기반), Spark로 전처리·토픽·인덱싱 수행, Opensearch 연동 및 웹 검색 인터페이스 제공.

## B_project
- **설명**: 알고리즘 문제 풀이 및 코딩 테스트 자동화·기록을 위한 라이브러리와 도구 모음.
- **기술**: Python 중심(라이브러리·노트북), Jupyter Notebook, 참고 문서로 FastAPI/Async DB/Redis 기반 백엔드 스택.
- **특징**: boj_bible(자료구조·알고리즘 모듈), CT 자동화 도구 및 LLM 호출유틸(기본 모델 변경 안내 포함), kakao_history.ipynb 등의 문제 풀이 예제 포함.
