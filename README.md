## geo_map
- **설명**: 좌표 또는 사각형 영역을 대한민국(남한/북한) 행정구역으로 빠르게 변환하는 도구  
- **기술**: Python, Shapely/STRtree(공간 인덱스), GeoJSON, GeoPandas 기반 데이터 처리  
- **특징**: 사각형→행정구역(대표/상세) 변환(고성능 v2), 섬 자동 감지 및 방향 기반 대표 표기, GeoJSON 생성·경계 조회·일괄 변환 지원

## Mini-Pipeline
- **설명**: YouTube 스크립트 추출부터 전처리·Kafka·Opensearch 업로드까지의 Docker 기반 스트리밍 E2E 파이프라인  
- **기술**: Docker Compose, Spark Structured Streaming, Kafka(kRaft), Opensearch, FastAPI + Nginx, Kiwipiepy(한국어 전처리)  
- **특징**: Spark→Kafka→Opensearch 흐름의 스트리밍 ETL, FastAPI/Nginx 기반 검색 UI 연결, Kiwipiepy 기반의 한국어 전처리(pandas_udf) 및 도커화된 구성

## B_project
- **설명**: 알고리즘 연습 및 코딩 테스트 자동화·도구 모음 저장소  
- **기술**: Python, Jupyter Notebook, 알고리즘 라이브러리, LLM 통합 유틸리티  
- **특징**: boj_bible(자료구조·그래프·트리·문자열·고급 알고리즘) 모듈, CT 자동화 도구 및 LLM 헬퍼(기본 모델 gpt-5-nano로 변경 안내), kakao_history.ipynb 등 문제 풀이 예제 추가
