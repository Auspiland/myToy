## geo_map
- **설명**: 사각형 영역이나 좌표를 대한민국/북한 행정구역(시/도 등)으로 변환해 대표문구와 포함/완전 포함 지역 정보를 반환합니다.  
- **기술**: Python 기반, Shapely/GeoJSON 이용한 공간처리, STRtree 공간 인덱스(성능 최적화).  
- **특징**: 사각형→행정구역 변환 및 대량 변환 지원, 섬 자동 감지·대표 표현 규칙, GeoJSON 생성 및 경계 조회 기능.

## Mini-Pipeline
- **설명**: YouTube URL로부터 스크립트를 추출해 문장 전처리 후 Kafka와 Opensearch로 업로드하는 스트리밍 E2E 파이프라인입니다.  
- **기술**: Docker(컨테이너화), Apache Spark(Streaming), Kafka(kRaft), Opensearch, FastAPI/Nginx(웹), Kiwipiepy(한국어 전처리).  
- **특징**: Streaming 기반 파이프라인(폴더 감시→Kafka 토픽→Opensearch 인덱싱), Kiwi 기반 오탈자·문장분리 처리, 컨테이너 네트워크 및 배포 구성 포함.

## B_project
- **설명**: 알고리즘 문제 풀이·코딩 테스트 연습을 위한 라이브러리 및 도구 모음입니다.  
- **기술**: Python 중심(라이브러리·노트북), Jupyter 노트북, LLM 유틸(README 내 GPT 호출 유틸 포함).  
- **특징**: boj_bible(자료구조·알고리즘 모듈) 구성, CT(코딩 테스트) 자동화 도구 및 예제(노트북의 문제 풀이 함수), LLM 관련 유틸 및 운영 가이드 문서 포함.
